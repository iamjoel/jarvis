# 人体免疫器官地图

## 高密度摘要

| 维度 | 内容 |
|------|------|
| 一句话结论 | 免疫器官不是单个“免疫力中心”，而是一套让免疫细胞出生、训练、巡逻、相遇、激活、记忆和退场的空间网络。 |
| 核心机制 | 骨髓生产免疫细胞，胸腺训练 T 细胞识别自我与非我，淋巴结监控组织液，脾脏监控血液，黏膜相关淋巴组织监控呼吸道、消化道和泌尿生殖道入口。 |
| 判断入口 | 想理解免疫细胞从哪里来，看骨髓；想理解 T 细胞自我耐受，看胸腺；想理解淋巴结肿大，看淋巴结；想理解脾切除感染风险，看脾脏；想理解肠道免疫和疫苗黏膜免疫，看 MALT/GALT。 |
| 常见误区 | 扁桃体、阑尾、脾脏都不是“没用器官”；淋巴结肿大不等于癌症；肠道是重要免疫界面，但不是唯一免疫器官；免疫器官强弱不能靠单一保健品判断。 |
| 相关文档 | [人体免疫细胞地图](./immune-cells.md)、[过敏深度指南](../a/allergy.md)、[自身免疫疾病深度指南](../z/zishen-mianyijibing.md)、[《肠子知道答案》](../b/book-gut-knows-the-answer.md) |

> 本文是免疫器官的机制地图，不用于自我诊断。淋巴结肿大、脾大、白细胞异常、反复感染、发热盗汗体重下降等情况，需要结合病程、体征、影像、血液检查和医生判断。

## 一、免疫器官可以分成两大类

| 类型 | 代表器官或组织 | 主要任务 |
|------|------|------|
| 中枢免疫器官 / 初级淋巴器官 | 骨髓、胸腺 | 生成免疫细胞，让 B 细胞和 T 细胞成熟，并筛掉明显危险的自身反应性细胞 |
| 外周免疫器官 / 次级淋巴器官 | 淋巴结、脾脏、扁桃体、腺样体、派尔集合淋巴结、阑尾、黏膜相关淋巴组织、皮肤相关免疫组织 | 让免疫细胞巡逻、相遇、接收抗原、被激活、形成效应反应和记忆 |

一个实用类比：

```text
骨髓：生产基地
胸腺：T 细胞训练学校
淋巴结：各地安检站，检查组织液来的线索
脾脏：血液过滤站，检查血液里的线索
黏膜相关淋巴组织：边境口岸，守住呼吸道和消化道入口
皮肤免疫系统：外墙和前哨
```

## 二、骨髓：免疫细胞的生产基地

骨髓是造血干细胞所在的核心场所。红细胞、血小板和大多数白细胞都源自骨髓中的造血干细胞。免疫系统里常见的中性粒细胞、单核细胞、嗜酸性粒细胞、嗜碱性粒细胞、B 细胞、NK 细胞等，都与骨髓造血密切相关。

| 维度 | 要点 |
|------|------|
| 核心任务 | 造血，产生免疫细胞前体和多种成熟白细胞 |
| 和免疫细胞的关系 | B 细胞在骨髓成熟；T 细胞前体来自骨髓，但需要去胸腺成熟；髓系细胞主要由骨髓持续供应 |
| 典型相关问题 | 白细胞减少、贫血、血小板异常、白血病、骨髓抑制、化疗后免疫低下 |
| 常见误解 | “白细胞少”不是一个单一疾病名，可能是生产减少、消耗增加、药物影响、感染、免疫病或血液病导致 |

骨髓的重要性在于：免疫防御不是只靠现有细胞“更努力”，还要靠持续生产足够、合格、分化方向正确的新细胞。

## 三、胸腺：T 细胞的训练和筛选学校

胸腺位于胸骨后方，在儿童和青春期相对活跃，成年后逐渐萎缩但并非完全无功能。T 细胞前体从骨髓迁入胸腺，在这里接受选择，学习如何识别 MHC 分子，同时避免过度攻击自身组织。

| 过程 | 含义 | 结果 |
|------|------|------|
| 阳性选择 | 能适度识别自身 MHC 的 T 细胞被保留 | 确保 T 细胞能读懂人体细胞展示的抗原 |
| 阴性选择 | 对自身抗原反应太强的 T 细胞被删除或转为调节性 T 细胞 | 降低自身免疫风险 |
| 分化成熟 | T 细胞形成 CD4+、CD8+ 等方向 | 进入外周执行辅助、杀伤、调节和记忆功能 |

| 维度 | 要点 |
|------|------|
| 核心任务 | 训练 T 细胞，建立中枢耐受 |
| 典型相关问题 | 胸腺发育异常、重症肌无力相关胸腺异常、年龄增长后的 T 细胞新生能力下降 |
| 常见误解 | 胸腺不是普通意义上的“内分泌补品靶点”；它的核心价值是 T 细胞教育和耐受 |

胸腺解释了一个关键问题：免疫系统不是只要能攻击外敌就行，还必须学会“不攻击自己”。

## 四、淋巴管和淋巴：把组织里的线索送回检查站

淋巴系统负责回收组织间液，并把组织里的抗原、细胞碎片、免疫细胞和炎症信号带到淋巴结。它像一个连接外周组织和免疫检查站的排水与情报网络。

| 组成 | 作用 |
|------|------|
| 淋巴毛细管 | 从组织间隙收集多余液体和抗原信息 |
| 淋巴管 | 把淋巴液送往淋巴结，最终回流血液 |
| 淋巴液 | 携带抗原、免疫细胞、脂肪吸收产物和组织信号 |

淋巴系统和血液循环不同：血液是高速主干道，淋巴更像从组织边缘慢慢收集信息的支路系统。

## 五、淋巴结：组织液的免疫检查站

淋巴结沿淋巴管分布，常见于颈部、腋窝、腹股沟、肠系膜等区域。它们过滤来自局部组织的淋巴液，让树突状细胞、T 细胞和 B 细胞在这里相遇。

| 区域 | 主要细胞 | 功能 |
|------|------|------|
| 皮质滤泡 | B 细胞、滤泡树突状细胞 | B 细胞识别抗原，形成生发中心，产生高质量抗体反应 |
| 副皮质区 | T 细胞、树突状细胞 | T 细胞被抗原呈递细胞激活 |
| 髓质 | 浆细胞、巨噬细胞 | 抗体分泌、吞噬和过滤 |

| 维度 | 要点 |
|------|------|
| 核心任务 | 过滤组织液，启动局部适应性免疫 |
| 为什么会肿大 | 感染、炎症、疫苗接种、免疫病、肿瘤转移或血液系统疾病都可能导致淋巴结增大 |
| 判断入口 | 位置、大小、质地、疼痛、活动度、持续时间、是否伴随发热盗汗体重下降，都比单看“有没有肿”更重要 |
| 常见误解 | 疼痛性淋巴结常见于感染炎症；无痛并持续增大的淋巴结更需要警惕，但不能凭手感自行定性 |

淋巴结的关键价值是让“稀有的特异性 T/B 细胞”更容易遇到“对应抗原”，否则免疫系统很难在全身海量细胞中快速匹配目标。

## 六、脾脏：血液的免疫过滤站

脾脏位于左上腹，是最大的外周淋巴器官之一。它过滤血液，而不是过滤淋巴液；因此特别擅长处理血液中的病原体、衰老红细胞和带包膜细菌相关风险。

| 区域 | 主要任务 |
|------|------|
| 白髓 | 富含 T/B 细胞，监控血液抗原，启动适应性免疫 |
| 红髓 | 清除衰老或异常红细胞，吞噬血液中的颗粒和微生物 |
| 边缘区 | 捕捉血液中抗原，快速启动 B 细胞反应 |

| 维度 | 要点 |
|------|------|
| 核心任务 | 过滤血液、清除老化红细胞、处理血源性抗原、支持抗体反应 |
| 典型相关问题 | 脾大、脾功能亢进、外伤性脾破裂、脾切除后感染风险增加 |
| 特别重要 | 无脾或脾功能低下者，对肺炎链球菌、脑膜炎球菌、流感嗜血杆菌等带包膜细菌风险更高 |
| 常见误解 | 脾脏不是可有可无；虽然人可以在无脾状态下生活，但感染风险管理会改变 |

脾脏解释了为什么有些人脾切除后需要更重视疫苗、感染预警和医生给出的预防策略。

## 七、扁桃体和腺样体：咽喉入口的免疫哨所

扁桃体和腺样体位于口咽、鼻咽入口附近，属于黏膜相关淋巴组织的一部分。它们接触空气、食物和口鼻进入的微生物，帮助局部免疫系统识别常见抗原。

| 组织 | 位置 | 功能 |
|------|------|------|
| 腭扁桃体 | 口咽两侧，日常最常说的“扁桃体” | 监控口咽入口抗原 |
| 咽扁桃体 / 腺样体 | 鼻咽顶部 | 监控鼻腔后方入口，儿童期较活跃 |
| 舌扁桃体 | 舌根 | 参与咽部免疫防御 |

| 典型相关问题 | 关键理解 |
|------|------|
| 扁桃体反复发炎 | 既可能是局部感染反复，也可能与儿童上呼吸道暴露频繁有关 |
| 腺样体肥大 | 可能影响鼻塞、打鼾、张口呼吸和中耳问题 |
| 扁桃体切除 | 是否切除取决于反复感染频率、阻塞程度、并发症和医生评估，不是因为它“没用” |

## 八、黏膜相关淋巴组织：守住最大面积的入口

人体和外界接触面积最大的地方不是皮肤，而是消化道、呼吸道和泌尿生殖道黏膜。黏膜相关淋巴组织（MALT）负责在这些入口处平衡两件事：对病原体迅速反应，对食物、共生菌和无害物质保持耐受。

| 类型 | 位置 | 代表结构 |
|------|------|------|
| GALT | 肠道相关淋巴组织 | 派尔集合淋巴结、阑尾、肠系膜淋巴结、孤立淋巴滤泡 |
| BALT | 支气管相关淋巴组织 | 呼吸道相关淋巴组织，成人平时较少，炎症刺激后可形成或增强 |
| NALT | 鼻咽相关淋巴组织 | 扁桃体、腺样体等 |

黏膜免疫的关键词是 IgA。分泌型 IgA 能在黏膜表面中和病原体和毒素，同时相对温和，减少不必要的强炎症。

## 九、肠道免疫组织：为什么肠道常被称为重要免疫器官

肠道每天接触大量食物抗原、微生物和代谢产物，因此拥有庞大的免疫细胞和淋巴组织网络。它既要防御病原体，又要容忍食物和正常菌群。

| 结构 | 作用 |
|------|------|
| 派尔集合淋巴结 | 小肠中的淋巴滤泡集合，采样肠腔抗原，启动黏膜免疫 |
| M 细胞 | 把肠腔中的抗原递送给下方免疫细胞 |
| 肠系膜淋巴结 | 汇集肠道抗原信息，参与免疫激活和耐受 |
| 阑尾 | 富含淋巴组织，参与肠道免疫和菌群生态，可能作为有益菌储备环境之一 |
| 固有层免疫细胞 | 包含浆细胞、T 细胞、巨噬细胞、树突状细胞等，维持局部防御与耐受 |

| 常见误区 | 更准确的理解 |
|------|------|
| “肠道是最大免疫器官，所以所有免疫问题都从肠道治” | 肠道确实是重要免疫界面，但免疫问题还涉及骨髓、胸腺、淋巴结、脾脏、皮肤、肺、遗传和全身炎症网络 |
| “阑尾没用” | 阑尾不是生命必需器官，但它含有淋巴组织，参与肠道免疫和菌群相关功能 |
| “菌群越多越好” | 更关键的是多样性、生态稳定性、屏障完整性和是否产生有益代谢物 |

## 十、皮肤免疫系统：外墙、传感器和局部免疫前线

皮肤不只是物理屏障，也是免疫器官。角质层、皮脂、抗菌肽、皮肤菌群、朗格汉斯细胞、树突状细胞、肥大细胞、T 细胞和巨噬细胞共同构成皮肤免疫系统。

| 组成 | 作用 |
|------|------|
| 角质层和紧密连接 | 限制病原体和刺激物进入 |
| 抗菌肽和皮脂 | 抑制部分微生物生长 |
| 皮肤菌群 | 占据生态位，帮助调节局部免疫 |
| 朗格汉斯细胞和树突状细胞 | 捕捉抗原并传递给 T 细胞 |
| 肥大细胞和 T 细胞 | 参与过敏、炎症、感染防御和组织修复 |

皮肤免疫解释了为什么屏障受损时，湿疹、接触性皮炎、感染和过敏样反应更容易反复。

## 十一、免疫器官如何协作：一次局部感染的空间流程

1. 病原体从皮肤、呼吸道或肠道黏膜进入。
2. 局部屏障细胞、巨噬细胞、肥大细胞和树突状细胞先识别危险信号。
3. 中性粒细胞等先天免疫细胞从血液赶到现场。
4. 树突状细胞带着抗原沿淋巴管进入附近淋巴结。
5. 淋巴结中 T 细胞和 B 细胞被激活并扩增。
6. 效应 T 细胞、抗体和其他免疫细胞回到感染部位清除病原体。
7. 如果病原体进入血液，脾脏会参与过滤和免疫应答。
8. 感染结束后，炎症退场，部分 T/B 细胞形成记忆。

## 十二、把免疫器官和常见问题对应起来

| 问题 | 重点器官或组织 | 关键理解 |
|------|------|------|
| 白细胞异常 | 骨髓、血液、脾脏 | 可能涉及生产、释放、消耗、分布或血液系统疾病 |
| 淋巴结肿大 | 淋巴结、局部感染区域 | 常见于感染炎症，也可能与肿瘤或免疫病有关，持续或伴全身症状需就医 |
| 反复上呼吸道感染 | 扁桃体、腺样体、呼吸道黏膜、淋巴结 | 儿童暴露增加很常见，但严重、持续或异常感染需排查免疫缺陷 |
| 肠道炎症和食物相关反应 | GALT、肠系膜淋巴结、肠屏障 | 要区分感染、过敏、不耐受、自身免疫和功能性问题 |
| 脾切除后感染风险 | 脾脏 | 血液中过滤带包膜细菌的能力下降，需要遵医嘱管理疫苗和感染风险 |
| 疫苗接种 | 肌肉/皮肤局部、树突状细胞、淋巴结、脾脏 | 疫苗抗原通常需要被抗原呈递细胞带到淋巴组织，才能形成 T/B 细胞记忆 |
| 自身免疫 | 胸腺、骨髓、淋巴结、靶器官 | 自我耐受建立和外周免疫刹车失效都可能参与 |

## 十三、最小知识框架

如果只记一张图，可以这样理解：

```text
骨髓生产免疫细胞
  ↓
胸腺训练 T 细胞，降低误伤自己
  ↓
血液和淋巴把细胞与抗原送到各处
  ↓
淋巴结检查组织来的线索
  ↓
脾脏检查血液里的线索
  ↓
皮肤、呼吸道、肠道黏膜守住入口
  ↓
免疫反应结束后形成记忆，并让炎症退场
```

免疫系统不是一个器官，而是一张分布式网络。理解免疫器官，就是理解免疫细胞在哪里被制造、在哪里受训、在哪里相遇、在哪里被激活、又在哪里执行任务。
