## 历史背景

这一时期是中国历史上最为分裂动荡的时代之一，长期的战乱、民族融合与政权更迭，使官制出现了**多元并进、南北各异**的复杂局面。

核心特征：
- **门阀制度兴盛**：九品中正制将官职与门第深度绑定，形成"上品无寒门，下品无势族"的格局。
- **军事化倾向强烈**：武将与军事机构地位空前，都督制成为支配政治的关键。
- **官制频繁更迭**：各政权短命，官职名称与职能变动频繁，难以统一梳理。
- **少数民族政权的汉化**：北方少数民族政权（十六国、北朝）大量汉化官制，反向推动了官制融合。

## 三国时期

### 曹魏（220—265年）

曹魏基本继承汉制，但进行了重要调整：

#### 中央官制

| 官职 | 职能 | 特点 |
|------|------|------|
| **皇帝** | 最高统治者 | |
| **三公（太尉、司徒、司空）** | 最高荣誉职，实权已虚化 | 曹魏三公沦为荣誉衔 |
| **尚书令** | 实际行政中枢，相当于首相 | 尚书台（尚书省雏形）地位上升 |
| **中书令/中书监** | 新设，掌皇帝诏令起草 | 曹魏新设中书省，分尚书台的文书权力 |
| **侍中** | 皇帝顾问，参与决策 | |
| **九卿** | 沿袭汉制 | |
| **相国（丞相）** | 曹操历任魏王时的实际权力中枢 | 曹操以丞相→魏公→魏王掌控汉政权 |

#### 军事官制（曹魏尤重）

| 官职 | 职能 |
|------|------|
| **大将军** | 最高军事统帅（曹真、曹爽、司马懿均任此职）|
| **太傅** | 辅政大臣（司马懿以太傅执政）|
| **都督中外诸军事** | 总揽全国军事的特加衔，实际统帅 |
| **征东/征西/征南/征北将军** | 四方战区指挥官 |

#### 地方官制

- 州：**都督**（军事）+ **刺史**（民政）分立，都督地位高于刺史
- 郡：太守
- 县：县令/县长

> 曹魏的**都督制**是这一时期最重要的官制创新，以军事区管辖若干郡，是后世藩镇制度的前身。

### 蜀汉（221—263年）

蜀汉以延续汉统为旗帜，官制几乎完全照搬东汉：

| 官职 | 代表人物 | 备注 |
|------|---------|------|
| **丞相** | 诸葛亮 | 蜀汉丞相权力极大，诸葛亮录尚书事，总揽军政 |
| **大将军** | 蒋琬、费祎 | 后期的实际执政者 |
| **尚书令** | 李严等 | 名义辅政 |
| **九卿** | 沿袭汉制 | |

### 孙吴（229—280年）

孙吴官制较为特殊，军权极重：

| 官职 | 特点 |
|------|------|
| **丞相** | 权力不及蜀汉丞相，受制衡 |
| **上大将军、大将军** | 军事最高官职，地位重要 |
| **领兵制度** | 将领私领部曲，军队私人化程度高 |

## 两晋时期

### 西晋（265—316年）

西晋恢复了一定程度的集权，但门阀势力大幅上升：

#### 中央官制

| 官职 | 职能 |
|------|------|
| **三公（太尉、司徒、司空）** | 荣誉职，无实权 |
| **太宰、太傅、太保** | 三师，最高荣誉职 |
| **尚书省（尚书令/仆射）** | 实际行政中枢 |
| **中书省（中书监/令）** | 起草诏命 |
| **门下省（侍中）** | 封驳审议诏令（雏形）|
| **九卿** | 沿袭旧制 |

> **三省雏形**：西晋时期，尚书省、中书省、门下省（侍中所在）三省格局初步形成，为隋唐三省六部制奠定了基础。

#### 地方官制

西晋恢复了**分封制**（封宗室为王），各王拥有军队与封地，直接导致"**八王之乱**"（291—306年），最终引发西晋灭亡。

### 东晋（317—420年）

东晋是门阀政治的高峰时期，形成了"**王与马，共天下**"的格局：

| 官职 | 实际权力者 | 说明 |
|------|-----------|------|
| **录尚书事** | 王导、谢安等门阀领袖 | 掌握实际行政权 |
| **大将军/太尉** | 王敦、桓温等强藩 | 掌握军事权 |
| **丞相/司徒** | 门阀成员轮任 | 荣誉职 |
| **皇帝** | 司马氏 | 实际权力微弱 |

东晋政治的本质是**皇权与门阀共治**：皇帝提供合法性，门阀（琅琊王氏、颍川庾氏、谯郡桓氏、陈郡谢氏）提供实际行政能力。

## 南北朝时期

### 南朝（宋、齐、梁、陈，420—589年）

南朝官制基本延续东晋，但皇权逐渐恢复，门阀势力衰落：

| 官职类型 | 说明 |
|---------|------|
| **中书省、尚书省、门下省** | 三省格局稳定化 |
| **领军将军** | 主管禁军，地位重要 |
| **中领军、中护军** | 禁军核心将领 |
| **荆州/扬州刺史** | 最重要的地方官，历代南朝皆以此为权力跳板 |

南朝后期，**寒人（寒门出身者）**通过担任皇帝近侍（典签等职）崛起，进一步打击门阀。

### 北朝（北魏、北齐、北周，386—581年）

#### 北魏的汉化改革（孝文帝，471—499年）

北魏孝文帝大力推行汉化：

| 改革内容 | 具体措施 |
|---------|---------|
| **官制汉化** | 废除鲜卑旧官制，全面采用汉式三省九卿体系 |
| **班禄制** | 建立官员薪俸制度，规范官僚体系 |
| **门第制度** | 确立鲜卑贵族与汉人士族的门阀等级 |

#### 北齐（550—577年）

北齐官制高度发达，三省分工明确：
- **中书省**：起草诏令
- **门下省**：审核封驳
- **尚书省**：行政执行（六部制最早在北齐基本成型）

#### 北周（557—581年）

北周宇文氏推行**《周礼》复古**，以六官体系为蓝本改革官制：

| 六官 | 长官 | 现代类比 |
|------|------|---------|
| 天官（冢宰）| 大冢宰 | 行政总长 |
| 地官（司徒）| 大司徒 | 民政部长 |
| 春官（宗伯）| 大宗伯 | 礼部长官 |
| 夏官（司马）| 大司马 | 军事长官 |
| 秋官（司寇）| 大司寇 | 司法长官 |
| 冬官（司空）| 大司空 | 工程长官 |

北周六官改革虽然复古色彩浓厚，但其整合管理职能的思路影响了隋朝官制设计。

## 选官制度：九品中正制

九品中正制由曹魏吏部尚书陈群创立（220年），是这一时期最重要的选官制度：

### 制度内容

| 环节 | 说明 |
|------|------|
| **大中正/中正官设立** | 各州郡设中正官，由本籍高级官员担任 |
| **品评人才** | 中正官将本籍士人按家世与才德分为九品（上上、上中、上下、中上、中中、中下、下上、下中、下下）|
| **授官依据** | 吏部依据品评结果授予官职，品高者授高官 |

### 演变

| 时期 | 特点 |
|------|------|
| 曹魏初期 | 兼顾家世与才德，相对公平 |
| 西晋以后 | 中正官几乎全由门阀把持，品评几乎完全以门第决定 |
| 南北朝 | "上品无寒门，下品无势族"，彻底蜕变为门阀垄断官途的工具 |

## 优缺点分析

### 优点

- **三省制度的发展**：这一时期三省（尚书省、中书省、门下省）逐渐走向成熟，为隋唐完善三省六部制奠定了基础。
- **都督制的贡献**：以军事区划管辖地方，在战乱时期提高了局部行政效率。
- **北魏汉化改革**：促进了民族融合，将鲜卑等少数民族整合入中华官僚文明体系，对中国多民族国家的形成有深远意义。
- **班禄制推广**：官员薪俸制度化，减少了随意搜刮，有利于廉洁行政。

### 缺点

- **九品中正制的蜕化**：初衷是选贤任能，实际上成为门阀世家垄断官职的工具，严重压制了社会流动性。
- **门阀政治的弊端**：皇权软弱，门阀势大，政治决策受少数家族控制，缺乏广泛社会代表性。
- **军事化导致官制混乱**：都督、将军等军职远比文官职位重要，文武失衡。
- **官制频繁变动**：各政权短命，官制随政权更迭而反复更改，缺乏稳定性，行政成本极高。
- **中央权威长期缺失**：无论南朝还是北朝，皇权都难以完全驾驭地方军事势力，割据倾向始终存在。

## 历史地位与影响

三国两晋南北朝是中国官制史上的**过渡与转型期**：
1. **三省制度成型**：尚书、中书、门下三省在这一时期由雏形走向成熟，直接影响隋唐三省六部制。
2. **六部雏形出现**：北齐六部分工基本确立，成为隋唐六部的直接前身。
3. **九品中正制的历史教训**：其失败深刻揭示了门阀垄断选官的危害，直接推动了隋文帝创立科举制度的改革动力。
4. **少数民族官制融合**：北朝政权的汉化为隋唐多元一体的官制体系提供了重要素材。
