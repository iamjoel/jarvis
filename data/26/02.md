### 🦫 [Happycapy](https://happycapy.ai/app)
Happycapy 是一台面向所有人设计的 Agent 原生计算机，它包含三个核心部分：

- 由 Claude Code 和 MiniMax Agent 驱动的“计算核心”
- 为每位用户独立分配的安全云端运行环境（sandbox）
- 一个真正用户友好的 GUI 操作系统

不同于 🦞 的复杂部署和高门槛，
HappyCapy 开箱即用，无需安装，无需配置，也无需担心安全问题。
无论是在浏览器，还是手机端，你都可以随时随地运行 Agent。
内置 Agent 邮件系统、精选 Skill Store、可视化 Agent Teams，以及自动化定时任务。

定时任务(Automation) 和 Agent Team(Claude Code 那) 是付费才能用。

能力和 Manus 相当。 Manus 现在也是支持 Skills 了。
