# 肾脏深度指南

> **核心主旨**：肾脏是人体最精密的"化工厂"——每分钟过滤约 1.2 升血液，维持体液、电解质与酸碱的精确平衡，同时兼任内分泌器官，调控血压、红细胞生成与钙代谢。肾脏功能一旦损伤往往悄无声息，待发现时常已不可逆；理解肾脏的工作原理、早期预警信号与保护策略，是守护全身健康的基础。

---

## Q1：肾脏在哪里、长什么样？

**A：**  
人体有**两个**肾脏，位于腹腔后壁、脊柱两侧（腰部偏上），左肾略高于右肾（因右侧有肝脏占位）。每个肾脏：

- 外形似蚕豆，约 **10–12 cm 长、5–6 cm 宽、3–4 cm 厚**；
- 重量约 **120–170 g**；
- 内侧凹陷处称**肾门**，是肾动脉、肾静脉、输尿管和淋巴管的进出通道。

### 宏观结构：
| 结构 | 功能 |
|------|------|
| **肾皮质**（外层） | 含肾小球和部分肾小管，负责过滤和初步重吸收 |
| **肾髓质**（内层） | 含肾锥体（集合管和髓袢），负责尿液浓缩 |
| **肾盏 → 肾盂** | 收集尿液，汇入输尿管 |
| **肾被膜** | 保护外层，减缓冲击 |

## Q2：肾脏是如何工作的？一滴尿是怎么形成的？

**A：**  
肾脏的基本功能单位是**肾单位（Nephron）**，每个肾脏约含 **100 万个**肾单位。尿液生成分三个阶段：

### 第一步：超滤（Filtration）
- 血液进入**肾小球**（毛细血管球），血压将水、小分子溶质（葡萄糖、尿素、电解质）强制压过滤膜；
- 蛋白质、血细胞因分子太大留在血液中；
- 产生**原尿**（超滤液），每天约 **180 升**。

### 第二步：选择性重吸收（Reabsorption）
- 原尿流过近曲小管、髓袢、远曲小管；
- **99% 的水**和几乎全部葡萄糖、氨基酸被重新吸回血液；
- 电解质（Na⁺、K⁺、Ca²⁺）按需精确调节；
- 肾髓质高渗梯度促使水分进一步浓缩。

### 第三步：分泌（Secretion）
- 肾小管主动将血液中多余的 H⁺、K⁺、药物代谢产物、某些毒素分泌进管腔；
- 最终尿液仅约 **1.5–2 升/天**，含尿素、肌酐、多余电解质和代谢废物。

> **关键数据**：肾脏每分钟过滤约 **125 mL** 血浆（肾小球滤过率，GFR），每天循环过滤血液约 **50 次**。

## Q3：肾脏除了产生尿液，还有哪些功能？

**A：**  
肾脏是多功能器官，远不止"过滤废物"：

### 1. 电解质与体液平衡
- 精确调节血液中的 Na⁺、K⁺、Cl⁻、HCO₃⁻、Ca²⁺、磷酸盐浓度；
- 维持细胞内外液渗透压，保证细胞正常功能（心肌、神经尤其敏感）。

### 2. 酸碱平衡
- 通过排泄 H⁺ 和重吸收 HCO₃⁻，维持血液 pH 在 **7.35–7.45** 的极窄范围；
- 肾脏是慢性（数小时至数天）酸碱调节的主要器官（肺脏负责分钟级调节）。

### 3. 血压调控
- **肾素-血管紧张素-醛固酮系统（RAAS）**：血压或血容量下降时，肾脏分泌肾素，激活血管紧张素 II → 血管收缩 + 醛固酮促 Na 潴留 → 血压升高；
- 肾脏亦可通过排钠利尿来降低血容量和血压（"压力-利尿"机制）。

### 4. 红细胞生成调控
- 肾脏（主要是肾皮质成纤维细胞）分泌**促红细胞生成素（EPO）**；
- EPO 作用于骨髓，刺激红细胞前体分化；
- 肾病患者 EPO 分泌减少 → 肾性贫血，是慢性肾病的常见并发症。

### 5. 维生素 D 最终活化
- 皮肤合成的维生素 D₃ → 肝脏羟化为 25(OH)D → **肾脏再次羟化为活性 1,25(OH)₂D（骨化三醇）**；
- 骨化三醇促进肠道钙吸收、调节骨代谢；
- 肾功能下降 → 活性维生素 D 减少 → 钙磷代谢紊乱 → 继发性甲旁亢与肾性骨病。

### 6. 废物与毒素排泄
- 代谢废物：尿素（蛋白质代谢）、肌酐（肌肉代谢）、尿酸（嘌呤代谢）；
- 外源性物质：药物、造影剂、环境毒素；
- 约 **25–30%** 的药物主要经肾脏排泄，肾功能下降时需调整剂量。

## Q4：衡量肾脏功能的核心指标有哪些？

**A：**  
临床上最常用的肾功能指标：

### 一、肾小球滤过率（GFR / eGFR）
- **最核心**的肾功能指标，代表每分钟能过滤多少毫升血浆；
- 通常用血肌酐（Cr）+ 年龄 + 性别 + 种族估算（CKD-EPI公式）；
- 正常值：**≥90 mL/min/1.73m²**（年龄越大自然下降）。

| eGFR（mL/min/1.73m²） | CKD 分期 | 说明 |
|-----------------------|----------|------|
| ≥90 | G1（正常/高） | 合并肾损伤标志时为 CKD |
| 60–89 | G2（轻度下降） | 合并肾损伤标志时为 CKD |
| 45–59 | G3a（轻-中度） | 中等风险 |
| 30–44 | G3b（中-重度） | 高风险，需专科管理 |
| 15–29 | G4（重度） | 极高风险，准备肾替代治疗 |
| <15 | G5（肾衰竭） | 需透析或移植 |

### 二、尿白蛋白/肌酐比值（UACR）
- 正常：**<30 mg/g**（A1）；
- 微量蛋白尿：30–300 mg/g（A2），提示肾小球早期损伤；
- 大量蛋白尿：**>300 mg/g**（A3），显著增加 CKD 进展与心血管风险；
- UACR 升高早于 eGFR 下降，是早期肾损伤的敏感标志。

### 三、血肌酐（Cr）与血尿素氮（BUN）
- 血肌酐单独使用局限较大（受肌肉量影响，eGFR 下降 50% 后才明显升高）；
- BUN/肌酐比值 >20 提示肾前性因素（脱水、消化道出血）。

### 四、尿常规
- **蛋白尿**（++以上需复查+UACR）；
- **血尿**（镜下血尿：每高倍视野红细胞 >3 个，需排除肿瘤、结石、肾炎）；
- **管型**（颗粒管型、红细胞管型提示肾小球肾炎）。

## Q5：什么是慢性肾病（CKD）？它有多普遍？

**A：**  
**慢性肾病（Chronic Kidney Disease, CKD）** 定义为：肾脏结构或功能异常持续 **>3 个月**，表现为：
- eGFR <60 mL/min/1.73m²，或
- 存在肾损伤标志（蛋白尿、血尿、影像学异常、活检异常等），不论 eGFR 高低。

### 流行病学：
- 全球 CKD 患病率约 **9.1–13.4%**（不同研究和地区有差异）；
- 中国成人 CKD 患病率约 **10.8%**（2012 年 NEJM 大规模调查），估计患者超 **1.2 亿**；
- CKD 是**全球第九大死亡原因**，且随人口老龄化持续上升；
- 约 **50%** 的 CKD 患者不知道自己患病（早期无症状）。

### 主要病因（全球）：
| 病因 | 占比 |
|------|------|
| **糖尿病肾病** | 约 30–40%（第一大原因） |
| **高血压肾损害** | 约 25–30% |
| **肾小球肾炎** | 约 10–15% |
| **多囊肾** | 约 5% |
| 其他（狼疮性肾炎、肾盂肾炎、梗阻性肾病、药物性等） | 约 15–25% |

## Q6：CKD 早期有哪些症状？如何识别危险信号？

**A：**  
CKD 最危险的特点是**早中期几乎无症状**——肾脏代偿能力极强，直到 GFR 降至 30% 以下才出现明显不适。

### 早期可能出现的细微信号：
- **泡沫尿**（蛋白尿的视觉表现：尿液表面持久泡沫，似啤酒泡）；
- **夜尿增多**（肾脏浓缩功能下降，夜间频繁排尿）；
- **眼睑/脚踝轻度水肿**（白蛋白丢失导致胶体渗透压下降）；
- **血压升高**（尤其是年轻人出现难以控制的高血压）；
- **血常规提示轻度贫血**（EPO 不足）。

### 中晚期症状（GFR <30 时）：
| 系统 | 症状 |
|------|------|
| 全身 | 疲乏、虚弱、注意力下降（尿毒素积累） |
| 消化道 | 食欲减退、恶心、呕吐、口腔氨味 |
| 皮肤 | 瘙痒（钙磷沉积）、皮肤蜡黄（贫血+尿色素） |
| 心血管 | 心包炎、心力衰竭、高钾心律失常 |
| 骨骼 | 骨痛、骨折风险增加（肾性骨营养不良） |
| 神经 | 下肢不宁综合征、末梢神经病变、尿毒症脑病 |

> ⚠️ **红旗症状**（需紧急就诊）：
> - 尿量突然显著减少（急性肾损伤）
> - 严重水肿 + 气短（急性肺水肿）
> - 肌无力 + 心电图异常（高钾血症）
> - 血压骤升 + 剧烈头痛 + 视觉改变（高血压危象）

## Q7：急性肾损伤（AKI）是什么？与 CKD 有何不同？

**A：**  
**急性肾损伤（Acute Kidney Injury, AKI）** 是肾功能在 **48 小时内**的急性恶化：
- 血肌酐升高 ≥0.3 mg/dL，或
- 血肌酐在 7 天内升至基线的 1.5 倍以上，或
- 尿量 <0.5 mL/kg/h 持续 6 小时以上。

### AKI 三大分类：

| 类型 | 机制 | 常见原因 |
|------|------|---------|
| **肾前性**（最常见，约55%） | 肾脏灌注不足 | 脱水、心力衰竭、严重感染/休克、NSAID + ACEI 联用 |
| **肾性**（约40%） | 肾实质直接损伤 | 急性肾小管坏死（缺血/毒素）、急性肾小球肾炎、造影剂肾病、横纹肌溶解 |
| **肾后性**（约5%） | 尿路梗阻 | 前列腺增生、肾结石、肿瘤压迫 |

### AKI vs. CKD：

| 特征 | AKI | CKD |
|------|-----|-----|
| 时程 | 数小时至数天 | >3 个月 |
| 可逆性 | 多数可逆（若及时处理） | 多不可逆，进行性进展 |
| 关键危险 | 高钾血症、肺水肿、急性死亡 | 心血管并发症、进展至终末期肾病 |
| 相互关系 | AKI 反复发作显著增加 CKD 风险 | CKD 患者更易发生 AKI |

## Q8：糖尿病和高血压如何损伤肾脏？

**A：**  
这两者是 CKD 最主要的病因，损伤机制有所不同：

### 糖尿病肾病（Diabetic Nephropathy）

**核心机制：**
1. **血糖毒性**：高血糖激活多元醇通路、晚期糖基化终末产物（AGE）、PKC 通路，损伤肾小球内皮和足细胞；
2. **肾小球高滤过**：早期高血糖导致入球小动脉扩张，GFR 异常升高（>120 mL/min），加速肾小球损耗；
3. **肾小球基底膜增厚**：AGE 使基底膜胶原交联，通透性改变→蛋白漏出；
4. **系膜扩张与结节硬化**：Kimmelstiel-Wilson 结节是糖尿病肾病的病理标志；
5. **慢性缺氧与炎症**：最终导致肾间质纤维化。

**进程**：微量蛋白尿 → 大量蛋白尿 → GFR 下降 → 终末期肾病（历时 10–20 年）

### 高血压肾损害（Hypertensive Nephrosclerosis）

**核心机制：**
1. **血管壁损伤**：持续高压使肾入球/出球小动脉壁增厚、透明变性（动脉硬化），管腔狭窄；
2. **肾小球缺血**：灌注压不稳定→肾小球局灶节段硬化（FSGS）；
3. **RAAS 过度激活**：高血压→RAAS 激活→血管紧张素 II 直接收缩出球小动脉→肾小球内高压→进一步损伤；
4. **恶性高血压**（舒张压 >120 mmHg）可在数天内导致急性肾损伤。

> **双向关系**：肾病本身也会加重高血压（RAAS 过度激活 + 水钠潴留），形成恶性循环，是心血管-肾脏疾病（CvRKD）的核心机制。

## Q9：肾结石是怎么形成的？如何预防与治疗？

**A：**  

### 形成机制：
尿液中某些物质浓度超过溶解度，结晶聚集形成结石：

| 结石类型 | 占比 | 主要成因 |
|---------|------|---------|
| **草酸钙** | 约 75% | 高草酸饮食（菠菜、坚果）、低水分、高钠、低柠檬酸盐 |
| **磷酸钙** | 约 10% | 高尿钙、碱性尿（肾小管酸中毒） |
| **尿酸** | 约 10% | 高嘌呤饮食、痛风、酸性尿（pH <5.5） |
| **磷酸镁铵（鸟粪石）** | 约 5% | 慢性尿路感染（产尿素酶菌，如变形杆菌） |
| **胱氨酸** | <1% | 遗传性胱氨酸尿症 |

### 症状：
- **肾绞痛**：结石移动至输尿管，引发剧烈阵发性腰腹痛，可放射至腹股沟；
- 恶心、呕吐；
- 血尿（肉眼或镜下）；
- 尿频、尿急（结石接近膀胱）。

> ⚠️ 若合并发热（>38°C）+ 肾绞痛，提示梗阻性肾盂肾炎，需紧急处理（败血症风险）。

### 治疗选择：
| 方法 | 适应症 |
|------|--------|
| **保守治疗（多饮水+止痛）** | 直径 ≤5 mm，自行排出率约 85% |
| **药物助排（坦索罗辛）** | 5–10 mm 结石，输尿管下段，提高排出率约 20% |
| **体外冲击波碎石（ESWL）** | ≤2 cm 肾结石，首选非侵入性治疗 |
| **输尿管软镜碎石（URSL）** | 输尿管中下段结石，或 ESWL 失败后 |
| **经皮肾镜取石（PCNL）** | >2 cm 的复杂肾结石 |

### 预防（至关重要，复发率约 50%/5年）：
1. **大量饮水**：日尿量 >2.5 升（最有效，各类结石通用）；
2. **限盐**：高钠促进尿钙排泄（<6 g/天）；
3. **适量钙摄入**：不要过度限钙（饮食中钙可与草酸结合，减少尿草酸）；
4. **低动物蛋白**：减少尿酸、草酸和钙的排泄；
5. **增加柠檬酸摄入**（柠檬水）：柠檬酸盐抑制结石形成。

## Q10：什么是终末期肾病（ESRD）？透析是如何工作的？

**A：**  
**终末期肾病（ESRD）** 即 CKD G5 期（eGFR <15 mL/min/1.73m²），肾脏已无法维持生命所需的内环境平衡，需要**肾替代治疗（RRT）**：

### 一、血液透析（Hemodialysis）
- 每周 3 次，每次 3–4 小时在透析中心进行（或每日家庭透析）；
- 血液通过透析机的**半透膜**，废物和多余水分扩散至透析液中；
- 优点：清除效率高；缺点：依赖机器和固定时间，心血管波动大。

### 二、腹膜透析（Peritoneal Dialysis）
- 利用**腹膜**（自身的半透膜）每日自行在家进行；
- 通过腹腔导管注入透析液，废物缓慢弥散后引流；
- 优点：持续透析，血压更稳定，居家操作灵活；缺点：腹膜炎风险，长期有效性下降。

### 三、肾脏移植（最佳选择）
- 相比透析，移植患者**预期寿命提高约 10–15 年**（活体捐献效果优于尸体捐献）；
- 需长期服用免疫抑制剂（他克莫司 + 霉酚酸酯 + 糖皮质激素）；
- 主要障碍：器官来源不足（中国等待名单平均 3–5 年）。

### 透析患者的心血管风险：
- 透析患者心血管死亡率是普通人群的 **10–20 倍**；
- 透析本身不能替代肾脏的内分泌功能（EPO、维生素 D 活化需额外补充）。

## Q11：哪些习惯最容易损伤肾脏？

**A：**  
以下行为是肾脏的"隐形杀手"，很多人未意识到风险：

### 1. 长期滥用止痛药（NSAIDs）
- 布洛芬、双氯芬酸、阿司匹林等抑制前列腺素合成；
- 前列腺素维持肾脏微循环，长期使用→肾间质纤维化（镇痛药肾病）；
- **规则**：NSAIDs 不应连续使用超过 5–7 天，肾功能不全者禁用。

### 2. 长期过量使用含马兜铃酸的中草药
- 马兜铃酸（存在于关木通、广防己、天仙藤等）是强效肾毒素；
- 可导致**马兜铃酸肾病（AAN）**：进行性肾间质纤维化，快速进展至 ESRD；
- 同时是**尿路上皮癌**的确证致癌物（IARC 1类）。

### 3. 高盐饮食
- 高盐→血压升高→肾小球高压→肾损伤加速；
- 还促进蛋白尿和尿钙排出（结石风险）；
- WHO 推荐每日盐摄入 <5 g。

### 4. 高蛋白饮食（超量）
- 蛋白质代谢产生大量尿素、磷和氢离子，增加肾脏负担；
- 对健康人影响有限，但 CKD 患者应严格限制蛋白摄入（通常 0.6–0.8 g/kg/天）。

### 5. 长期脱水
- 尿液浓缩→结石形成风险↑；
- 急性大量脱水可直接引发肾前性 AKI。

### 6. 未控制的糖尿病和高血压
- 如 Q8 所述，这是 CKD 最主要的病因；
- **HbA1c <7%、血压 <130/80 mmHg** 是保护肾脏的核心目标。

### 7. 吸烟
- 尼古丁促进血管收缩，加速肾小球硬化；
- 吸烟者进展至 ESRD 的速度是不吸烟者的 **1.5–2 倍**。

### 8. 造影剂滥用（对比剂肾病）
- 碘对比剂（CT/冠造影剂）有肾毒性，高风险人群（已有 CKD、糖尿病、脱水）须术前水化并评估。

## Q12：如何科学保护肾脏健康？

**A：**  
肾脏保护是长期策略，核心在于**控制高危因素 + 定期筛查 + 避免肾毒素**：

### 一级预防（无 CKD 者）：

| 措施 | 目标/方式 |
|------|----------|
| **控血糖** | HbA1c <7%，避免血糖剧烈波动 |
| **控血压** | <130/80 mmHg；ACEI/ARB 类药物对糖尿病/蛋白尿患者有额外肾脏保护作用 |
| **戒烟** | 减少血管损伤 |
| **规律运动** | 中等强度有氧 150 分钟/周 |
| **健康饮食** | DASH 或地中海饮食；减少超加工食品和红肉 |
| **足量饮水** | 目标尿量 1.5–2 升/天（尿色淡黄为宜） |
| **谨慎用药** | 避免长期 NSAIDs、含马兜铃酸中药、不明成分保健品 |
| **定期体检** | 每年测血压、血糖；30 岁以上高危人群年度肾功能筛查 |

### 对 CKD 患者的额外措施：

| 措施 | 说明 |
|------|------|
| **ACEI/ARB 治疗** | 降低肾小球内压，减少蛋白尿，延缓进展（一线药物） |
| **SGLT-2 抑制剂** | 新型降糖药（达格列净、恩格列净）已证实在 CKD 中独立减少肾脏终点事件 |
| **限蛋白饮食** | 0.6–0.8 g/kg/天（在营养师监督下） |
| **控制血磷** | 限制含磷食品（加工食品、碳酸饮料）；磷结合剂（碳酸钙等） |
| **补充活性维生素 D** | 骨化三醇或帕立骨化醇 |
| **纠正贫血** | EPO 制剂 + 铁剂（目标 Hb 100–115 g/L） |
| **避免肾毒性药物/造影剂** | 用药前评估 GFR，调整剂量 |

### 筛查建议（高危人群）：

高危人群包括：糖尿病患者、高血压患者、60 岁以上老年人、有 CKD 家族史者、心血管病患者、长期使用肾毒性药物者。

建议每 **6–12 个月** 检测：
- 血肌酐 + eGFR；
- 尿常规 + UACR（晨尿）；
- 血钾、血磷（CKD G3+ 患者）。

## Q13：肾脏健康与心血管疾病有什么关系？

**A：**  
肾病与心血管病是**深度耦合**的双向关系，称为**心肾综合征（Cardiorenal Syndrome）**：

### 肾脏如何伤心：
- **容量负荷过重**：肾脏排钠障碍 → 水钠潴留 → 心脏前负荷增加 → 心力衰竭；
- **RAAS 过度激活**：血管紧张素 II → 高血压 + 心室肥厚；
- **交感神经过度激活**：肾缺血刺激交感 → 心率加快、心律失常；
- **炎症与氧化应激**：尿毒素（吲哚硫酸盐、对甲酚）直接损伤心血管内皮；
- **贫血**：高输出量负荷心脏；
- **钙磷代谢紊乱**：血管钙化（尤其冠状动脉和瓣膜）。

### 心脏如何伤肾：
- 心力衰竭 → 心输出量减少 → 肾灌注不足 → AKI 或 CKD 进展；
- 心房颤动 → 肾脏微栓塞；
- 心脏手术 → 造影剂 + 血流动力学波动 → AKI。

> **数据**：CKD 患者中约 **50–60%** 死于心血管事件（而非 ESRD），说明肾脏健康管理的核心目标之一就是心血管保护。

## 总结：行动清单

### 了解自己的肾脏状态：
- ✅ 是否有糖尿病、高血压或 CKD 家族史？
- ✅ 是否长期服用 NSAIDs 或不明中草药？
- ✅ 最近一次查过尿常规 + 肾功能（血肌酐/eGFR）是什么时候？

### 日常可做的事：
- ✅ 保持血压 <130/80 mmHg（有高血压者）；
- ✅ 控制血糖和 HbA1c（有糖尿病者）；
- ✅ 每天喝足水，保持尿色淡黄；
- ✅ 减少盐和超加工食品的摄入；
- ✅ 谨慎使用止痛药，避免长期连续服用；
- ✅ 戒烟；
- ✅ 定期体检（每年尿常规 + 肾功能）。

### 高危人群的医学管理：
- CKD G1–G2：强化危险因素控制，每年监测；
- CKD G3a–G3b：肾内科会诊，开始 ACEI/ARB 治疗，每 3–6 个月监测；
- CKD G4：肾内科主导，开始 ESRD 准备（透析教育、瘘管手术评估）；
- CKD G5：透析/移植评估，多学科团队管理。
