# Data

按“问题”组织内容。分类:

- 个人：人和自己的关系
- 社会：人和人类社会的关系
- 自然：人和自然的关系

建议的使用方式：

1. 先从问题进入，而不是先从作者、学科或文件夹进入。
2. 先看总问题，再顺着表格进入相关专题、拆书和资料。
3. 有价值的高频结论继续沉淀到 [knowledge](../knowledge/readme.md)。
4. 与解决问题有关的方法、自动化和 AI 工具统一放在文末附录。
