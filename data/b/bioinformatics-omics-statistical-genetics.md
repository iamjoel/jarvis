# 生物信息学、组学与统计遗传：生命科学为什么越来越像数据科学？

## 高密度摘要

**一句话结论**：现代生命科学已经从“观察少数基因和蛋白”进入“同时测量整个系统”的时代，基因组、转录组、蛋白质组、代谢组、表观组、微生物组和单细胞数据让生命科学越来越依赖统计和计算。**核心机制**：组学技术把生命系统拆成大规模数据矩阵，再通过统计模型寻找差异、关联、细胞类型、通路和风险模式；但数据越多，偏差、噪音、多重比较和解释错误也越多。**判断入口**：看到组学研究，先问测了什么层级、样本量多少、是否有验证队列、效应大小如何、能否解释因果。**常见误区**：测序不是读命运，GWAS 不是找到“决定基因”，单细胞图谱不是自动解释疾病，多组学整合也不等于因果证明。**相关文档**：[生命科学方法与模型生物](../l/life-science-methods-and-model-organisms.md)、[基因调控与表观遗传](../g/gene-regulation-and-epigenetics.md)、[生物医学证据判断](./biomedical-evidence-literacy.md)、[基因传](./book-the-gene.md)。

## 为什么生命科学进入数据时代？

过去的生命科学常常是“一次盯一个基因、一种蛋白、一个通路”。现在，测序和质谱等技术让科学家可以同时测量成千上万的分子。

这带来一个转变：

```text
单点观察 -> 系统测量 -> 数据建模 -> 生物学解释 -> 实验验证
```

现代生命科学越来越像数据科学，不是因为生命变成了数据，而是因为复杂系统需要大规模测量才能看见模式。

## 什么是“组学”？

“组学”指对某一类生命分子的整体测量。

| 组学 | 测什么 | 常回答的问题 |
------|--------|--------------|
| 基因组 | DNA 序列 | 有哪些遗传变异？ |
| 转录组 | RNA 表达 | 哪些基因正在被读取？ |
| 蛋白质组 | 蛋白质数量和修饰 | 哪些功能机器正在工作？ |
| 代谢组 | 小分子代谢物 | 代谢状态如何？ |
| 表观组 | DNA 甲基化、染色质开放等 | 哪些基因区域容易被读取？ |
| 微生物组 | 微生物组成和功能基因 | 群落里有哪些微生物，可能做什么？ |

这些层级不是互相替代，而是互相补充。DNA 是可能性，RNA 是表达状态，蛋白质和代谢物更接近实际功能。

## 基因组：不是命运书，而是风险和可能性地图

基因组测序能读出一个人的 DNA 序列。

它擅长：

- 诊断某些单基因遗传病
- 发现癌症驱动突变
- 指导部分药物选择
- 研究族群历史和亲缘关系
- 找到疾病相关遗传变异

它不擅长：

- 准确预测多数复杂疾病
- 判断性格、智力、命运
- 直接告诉你该怎么生活

多数常见疾病不是单个基因决定，而是许多微效变异与环境共同作用。

## GWAS：寻找变异和性状的统计关联

GWAS，全基因组关联研究，是把大量人的基因变异与某个性状或疾病进行统计比较。

它的基本逻辑：

```text
很多人基因型 + 很多人表型 -> 找哪些变异在某性状中更常见
```

GWAS 能发现关联位点，但要注意：

- 关联不等于因果。
- 找到的位点常常在非编码区。
- 单个位点效应通常很小。
- 结果依赖样本量和人群结构。
- 从位点到机制还需要大量实验。

GWAS 的价值不是“找到一个命运基因”，而是用统计方法揭示复杂性状的遗传结构。

## 遗传度：不是“百分之多少由基因决定”

遗传度常被误解。

遗传度说的是：在某个特定人群和环境中，个体差异有多少比例与遗传差异相关。

它不表示：

- 某个人的特征有多少由基因决定。
- 环境不重要。
- 特征不可改变。
- 换一个环境仍然一样。

比如身高遗传度很高，但营养和疾病仍能显著影响身高；在营养差异更大的环境中，环境贡献可能更大。

## 多基因风险评分：有用，但边界明显

多基因风险评分 PRS 是把许多遗传变异的小效应加总，估计某人某种疾病或性状的遗传倾向。

它可能有用：

- 辅助识别部分高风险人群
- 与家族史、生活方式和临床指标一起使用
- 做人群层面风险分层

它的限制：

- 对不同族群泛化能力有限。
- 不能替代临床诊断。
- 对个人命运预测能力常被夸大。
- 环境和行为仍然重要。

## 单细胞测序：从“组织平均值”看到细胞差异

传统转录组像把一锅汤搅匀后测味道。单细胞测序则尽量分辨每个细胞分别在表达什么。

它能回答：

- 组织里有哪些细胞类型？
- 疾病中哪些细胞状态改变？
- 免疫细胞是否被激活或耗竭？
- 肿瘤中有哪些不同克隆和微环境细胞？
- 发育过程中细胞命运如何转变？

但单细胞数据也有噪音：

- 测到的是 RNA，不一定等于蛋白功能。
- 细胞分离过程会改变状态。
- 聚类结果需要生物学解释。
- “发现新细胞类型”需要谨慎验证。

## 空间组学：细胞在哪里同样重要

单细胞测序告诉我们“有哪些细胞状态”，但常常丢失位置信息。空间转录组和空间蛋白组试图保留组织空间结构。

这很重要，因为生命不是散装细胞：

- 肿瘤边缘和中心细胞状态不同。
- 免疫细胞是否靠近癌细胞影响疗效。
- 发育中位置决定命运。
- 大脑功能依赖细胞在回路中的位置。

空间信息把“细胞清单”变成“组织地图”。

## 多组学：把多个层级连起来

单一组学常常只能看到一层。

多组学试图连接：

```text
DNA 变异 -> 染色质状态 -> RNA 表达 -> 蛋白功能 -> 代谢状态 -> 表型
```

这有助于理解疾病机制，但也更容易过拟合。多组学结果尤其需要独立验证和明确问题。

## 生物信息学不是“跑软件”

生物信息学的核心不是会用工具，而是把数据处理、统计假设和生物学问题对齐。

常见步骤：

1. 数据质控：去掉低质量样本和技术噪音。
2. 标准化：让样本可比较。
3. 差异分析：找显著变化。
4. 通路分析：看变化是否集中在某些功能模块。
5. 聚类和降维：寻找细胞类型或样本亚群。
6. 验证：用独立数据或实验验证。

每一步都有选择，每个选择都可能改变结果。

## 常见数据陷阱

| 陷阱 | 含义 |
------|------|
| 批次效应 | 样本处理时间、机器、试剂批次造成假差异 |
| 多重比较 | 同时测几万个指标，偶然显著不可避免 |
| 过拟合 | 模型记住训练数据，但无法预测新数据 |
| 样本量不足 | 复杂数据维度高，小样本很容易假阳性 |
| 人群结构 | 遗传研究中族群差异造成假关联 |
| 注释错误 | 把统计聚类过度解释为真实生物类别 |

## 如何读组学研究？

可以问 8 个问题：

1. 测的是哪一层组学？
2. 样本量和人群来源是什么？
3. 有没有独立验证队列？
4. 是否处理批次效应？
5. 是否做多重检验校正？
6. 结果是关联、预测还是机制？
7. 是否有实验验证？
8. 结论是否被说得超过数据能力？

## 一句话结论

组学让生命科学看见系统，但系统数据不是系统理解；真正的判断力在于知道每种数据测到了什么、没测到什么，以及从相关走向机制还差几步。
