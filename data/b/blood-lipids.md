# 血脂

## 高密度摘要

- **一句话结论**：血脂是一组衡量血液中胆固醇和甘油三酯运输状态的指标，其中 LDL-C、非 HDL-C、ApoB 和甘油三酯最常用于判断动脉粥样硬化和代谢风险。
- **核心机制**：血脂异常本质上是脂蛋白颗粒数量、成分和清除能力失衡，长期会推动动脉斑块形成。
- **判断入口**：体检发现 LDL-C、非 HDL-C、甘油三酯升高，或已有高血压、糖尿病、脂肪肝、家族早发心血管病时，应系统看血脂。
- **常见误区**：总胆固醇正常不等于风险低；HDL-C 高不能抵消高 LDL-C；饮食油腻不是血脂异常的唯一原因。
- **相关文档**：[动脉斑块深度指南](../d/dongmai-bankai.md)、[血栓深度指南](../x/xueshuan.md)、[健康索引](../health.md)。

## 一句话介绍

血脂是对血液中胆固醇、甘油三酯及其脂蛋白运输状态的度量，主要用于判断动脉粥样硬化、心梗和脑卒中风险。

## 异常值有什么问题

LDL-C、非 HDL-C 或 ApoB 长期偏高会增加动脉斑块形成风险，斑块破裂后可能导致心梗、脑卒中或外周动脉疾病。甘油三酯明显升高常提示胰岛素抵抗、脂肪肝或饮酒/饮食问题；极高甘油三酯还会增加急性胰腺炎风险。

## 正常值的范围

一般人群常用参考范围如下，单位为 mg/dL：

| 指标 | 理想/较好 | 需要关注 |
|------|------|------|
| 总胆固醇 TC | <200 | ≥200 |
| LDL-C | <100 | ≥130 更需关注；高危人群目标更低 |
| HDL-C | 男性 ≥40，女性 ≥50 | 低于上述范围提示风险增加 |
| 甘油三酯 TG | <150 | ≥150 |
| 非 HDL-C | <130 | ≥160 更需关注 |
| ApoB | 越低越好，常见目标 <90 或更低 | 高危人群目标更低 |
| Lp(a) | 多数人只需一生测一次 | 高水平主要由遗传决定 |

具体治疗目标取决于年龄、既往心血管病、糖尿病、慢性肾病、吸烟、高血压和总体 ASCVD 风险。

## 关联指标

- **血压**：共同决定心脑血管风险。
- **HbA1c / 空腹血糖**：糖代谢异常常伴随高甘油三酯和低 HDL-C。
- **腰围、BMI、脂肪肝指标**：反映胰岛素抵抗和代谢综合征。
- **ApoB、非 HDL-C、Lp(a)**：比单看 LDL-C 更能补充判断脂蛋白颗粒风险。
- **高敏 C 反应蛋白（hs-CRP）**：可作为炎症背景参考。

## 如何检测

- **血脂四项**：总胆固醇、LDL-C、HDL-C、甘油三酯；抽血，通常体检包含，单独检测常见约 30-100 元人民币。
- **ApoA1 / ApoB**：抽血，常见约 50-150 元人民币。
- **Lp(a)**：抽血，常见约 80-200 元人民币；因主要由遗传决定，通常无需频繁复查。
- **冠脉钙化积分（CAC）**：不是血脂检测，但可帮助中等风险人群判断是否已有动脉粥样硬化；价格和可及性差异较大。

是否需要空腹取决于检测项目和医院要求；甘油三酯受进食影响较明显。

## 如何恢复正常

常见导致异常的原因包括遗传、饱和脂肪和反式脂肪摄入过多、精制碳水和酒精过多、体重增加、胰岛素抵抗、甲状腺功能减退、肾病和某些药物。

改善方向：

1. 减少反式脂肪、过多饱和脂肪和超加工食品。
2. 增加可溶性膳食纤维、豆类、坚果、鱼类和橄榄油等更有利的脂肪来源。
3. 控制体重和腰围，改善胰岛素抵抗。
4. 规律有氧运动和力量训练。
5. 甘油三酯高时尤其要减少酒精和含糖饮料。
6. LDL-C、ApoB 或 Lp(a) 高且总体风险高时，应和医生讨论药物治疗，如他汀、依折麦布、PCSK9 抑制剂等。
