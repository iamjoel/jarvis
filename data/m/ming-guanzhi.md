## 历史背景

明太祖朱元璋出身贫苦，对元朝权相专政（如伯颜、脱脱等丞相主导政局）深怀戒惧。1380年，朱元璋以"谋反"罪杀宰相胡惟庸，随即**废除延续一千五百年的丞相制度**，将宰相的行政权力直接归于皇帝，开创了中国古代皇权专制的最高峰。

明朝官制的核心特征：
- **皇权至高无上**：废丞相后皇帝直接管理六部，权力集中达到顶峰。
- **内阁制度**：作为皇帝的秘书机构，逐渐演化为事实上的行政核心。
- **特务政治**：锦衣卫、东厂、西厂构成独特的皇帝特务系统，监控官民。

## 核心变革：废丞相与皇权独尊

### 废丞相的历史意义

| 时间 | 事件 |
|------|------|
| 1380年 | 朱元璋以胡惟庸案为由废丞相，规定"后代子孙不得再立丞相，臣下有请立者，处以极刑" |
| 废相后 | 皇帝直接统辖六部、五军都督府、都察院、大理寺、通政司（史称"皇帝总揽六部"）|

> 废丞相是中国两千年相权与皇权博弈的终结，标志着皇权专制达到顶点，但也意味着政务处理压力完全压在皇帝一人身上。

## 中央官制：内阁与六部

### 内阁

明朝内阁制度的演变：

| 时期 | 内阁性质 | 说明 |
|------|---------|------|
| **洪武时期** | 无内阁，皇帝直管六部 | 废丞相后朱元璋亲力亲为 |
| **永乐时期（1402年起）** | 内阁雏形 | 解缙等文官入文渊阁参与机务，称"内阁大学士" |
| **宣德以后** | 内阁地位上升 | 大学士开始"票拟"（草拟对奏章的处理意见）|
| **嘉靖后期** | 内阁首辅地位如同宰相 | 严嵩、徐阶、张居正等首辅实掌大权 |
| **万历时期** | 内阁权力顶峰 | 张居正以首辅身份推行全面改革 |

#### 内阁核心职能

| 职能 | 说明 |
|------|------|
| **票拟** | 在奏章上草拟处理意见（用小纸条附于奏章），供皇帝参考 |
| **顾问咨询** | 备皇帝顾问，参与重大政策讨论 |
| **起草诏令** | 代皇帝起草重要文件 |

> 内阁权力来源于皇帝的信任，而非制度授权，本质上仍是皇帝的秘书机构。皇帝勤政时（如洪武、永乐），内阁只是助手；皇帝怠政时（如嘉靖、万历），内阁实际主导朝政。

#### 内阁大学士官衔

| 殿阁名称 | 说明 |
|---------|------|
| 中极殿大学士（明初华盖殿大学士）| 首辅通常担任 |
| 建极殿大学士（明初谨身殿大学士）| 次辅 |
| 文华殿大学士 | 通常兼管詹事府 |
| 武英殿大学士 | |
| 文渊阁大学士 | |
| 东阁大学士 | |

> 大学士本为正五品，后加"尚书"衔升为正二品，实际影响力则取决于首辅地位。

### 司礼监（宦官机构）

| 职能 | 说明 |
|------|------|
| **批红** | 皇帝用红笔批阅奏章（"批朱"），当皇帝怠政时由司礼监秉笔太监代行 |
| **掌印太监** | 司礼监首领，掌皇帝御印，地位相当于内廷宰相 |

> 内阁"票拟"+ 司礼监"批红"的制度，客观上形成了内廷（宦官）与外廷（内阁）的双轨权力格局，但决定权始终在皇帝手中。这也给宦官专政（如王振、刘瑾、魏忠贤）提供了制度缝隙。

### 六部（直属皇帝）

废丞相后，六部直接对皇帝负责：

| 部名 | 尚书品级 | 主要职能 |
|------|---------|---------|
| **吏部** | 正二品 | 官员任免、考核、升降，被称为"天官" |
| **户部** | 正二品 | 户籍、赋税、财政 |
| **礼部** | 正二品 | 礼仪、教育、科举、外交礼宾 |
| **兵部** | 正二品 | 武官选用、军事行政（五军都督府管统兵，兵部管调兵）|
| **刑部** | 正二品 | 司法、刑狱 |
| **工部** | 正二品 | 工程、水利、屯田 |

### 都察院

- 最高监察机构，长官**左都御史、右都御史**（正二品）
- 下设十三道**监察御史**，负责分道监察地方

### 大理寺

- 最高复核机关，复审刑部的重大刑事判决
- "三法司"：刑部（初审）、都察院（监察）、大理寺（复核）共同处理重大案件

### 通政使司

- 负责接收和传递内外奏章，是信息通道枢纽

## 地方官制

### 两京十三布政使司（后增至十五）

明朝地方实行**布政使司—府—县**为主的三级制，另有都指挥使司（军事）和按察使司（司法）并立：

| 官职 | 机构 | 职能 |
|------|------|------|
| **布政使（左右布政使）** | 承宣布政使司 | 地方行政与财政 |
| **按察使** | 提刑按察使司 | 司法监察 |
| **都指挥使** | 都指挥使司 | 军事 |

> 三司（布政、按察、都指挥）分权制衡，防止地方官员集权，但协调效率低下。

### 巡抚与总督（中后期）

| 官职 | 性质 | 说明 |
|------|------|------|
| **巡抚** | 中央派驻地方的督察官 | 逐渐成为省级实际最高长官，综合协调三司 |
| **总督** | 统辖数省的军政长官 | 主要设于边境地区（如蓟辽总督），处理军事 |

### 府—县两级

| 级别 | 官职 |
|------|------|
| **府** | 知府（正四品）、同知、通判 |
| **县** | 知县（正七品）、县丞、主簿、典史 |

## 军事体系

明朝建立了**五军都督府 + 卫所制度**的军事体系：

### 五军都督府

| 机构 | 说明 |
|------|------|
| 中、左、右、前、后五军都督府 | 统辖全国卫所，但不直接调兵（发令权在兵部）|

> 五军都督府统兵而不调兵，兵部调兵而不统兵，相互制衡，防止武将专权。

### 卫所制度

| 单位 | 兵员 | 长官 |
|------|------|------|
| **卫** | 5600人 | 指挥使 |
| **所（千户所）** | 1120人 | 千户 |
| **百户所** | 112人 | 百户 |

卫所军人世代为军户，平时屯田，战时出征，类似隋唐府兵制，但后期军户逃亡严重，战斗力严重下降。

### 营兵制（中后期）

嘉靖以后，卫所制度废弛，明朝转向募兵制的**营兵**：
- 戚继光创立**戚家军**，为募兵制职业化军队的代表
- 督抚统领营兵，地方军事实权上移

## 监察体系

明朝监察制度的特点：

| 机构 | 职能 |
|------|------|
| **都察院** | 中央最高监察机关，御史弹劾百官 |
| **六科给事中** | 对应六部，监察六部官员，可封驳六部命令 |
| **巡按御史** | 代天子巡视地方，"以七品官监督二品抚台" |
| **厂卫** | 皇帝直接掌控的特务监察（见下节）|

> **六科给事中制度**是明朝监察的一大创新：六科对应六部，形成对六部的专项监察，是中国古代行政监督最精密的设计之一。

## 特务机构

明朝建立了中国历史上最完备的皇权特务体系：

| 机构 | 设立时间 | 职能 | 备注 |
|------|---------|------|------|
| **锦衣卫** | 洪武年间 | 皇帝亲军，兼负侦缉、逮捕、审讯官民 | 设有专用监狱"诏狱" |
| **东厂** | 永乐年间 | 宦官特务机构，监视锦衣卫与百官 | 由宦官掌管，权力凌驾锦衣卫之上 |
| **西厂** | 成化年间 | 进一步强化宦官特务体系 | 汪直主持，后被废除 |
| **内行厂** | 正德年间 | 刘瑾设立，监视东厂和西厂 | 后随刘瑾被诛而撤销 |

> 厂卫制度是明朝皇权专制的极端体现，官员和民众均生活在特务监视之下，严重摧残了正常的官僚文化和士大夫精神。

## 选官制度

### 科举制度的规范化

明朝对科举进行了最为严格的规范：

| 内容 | 说明 |
|------|------|
| **三级考试** | 院试（秀才）→ 乡试（举人）→ 会试+殿试（进士）|
| **八股文** | 考试文体固定为八股文格式，内容限于四书五经 |
| **翰林院庶吉士** | 进士中优者进入翰林院继续培养，是入阁的必经之路 |
| **非进士不入翰林，非翰林不入内阁** | 明代内阁成员几乎清一色翰林出身 |

## 优缺点分析

### 优点

- **皇权集中**：废丞相杜绝了历史上权相专政的弊病，皇帝对政务的掌控达到空前程度。
- **内阁票拟制度**：一定程度上弥补了废相后皇帝精力不足的问题，为君主怠政时提供了行政缓冲。
- **六科给事中制度**：对六部的专项监察极为精密，是明朝行政监督的重要成就。
- **科举制度完备**：三级考试体系严格规范，"进士—翰林—内阁"的晋升通道清晰，保证了高层官员的知识水平。

### 缺点

- **废丞相的副作用**：皇权过于集中，皇帝能力参差不齐时（如嘉靖、万历），政务陷入停滞；而勤政皇帝（如朱元璋）则身心俱疲。
- **宦官专政的制度性隐患**：批红制度给司礼监宦官提供了专权的入口，魏忠贤等宦官专政是明朝治理的最大痼疾。
- **厂卫特务政治**：严重压抑了官员的积极性和社会的创造力，催生了官场的保守与腐败。
- **八股文的弊端**：科举考试内容高度僵化，脱离实务，培养出大量文辞精美但缺乏行政能力的官员。
- **卫所制度废弛**：世袭军户制度后期严重败坏，军队战斗力极低，明朝后期无法有效抵御满洲入侵。
- **三司分权效率低**：布政、按察、都指挥三司协调困难，不得不以巡抚、总督补充协调，实际形成四级地方体系，行政层级过多。

## 历史影响

明朝官制最重要的遗产有两点：
1. **废丞相**的历史教训：集权过度最终导致皇权专制走向极端，清朝军机处的设立是对这一教训的进一步强化。
2. **内阁制度**：作为皇帝秘书团向实质性行政机构演化的历程，与近代责任内阁制有一定历史渊源，影响后世。
