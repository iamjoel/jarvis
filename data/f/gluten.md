# 麸质深度指南

> **核心概念**：麸质（Gluten）是小麦、大麦、黑麦等谷物中两类储存蛋白——醇溶蛋白（Gliadin）与谷蛋白（Glutenin）——遇水揉合后形成的三维蛋白网络。它赋予面团弹性与延展性，是人类几千年来烘焙技艺的物质基础。对于绝大多数人，麸质既是安全的食物成分，也是优质蛋白来源；但对特定遗传体质的人群，它却是触发严重自身免疫损伤的"扳机"。

---

## 一、麸质是什么？——从分子到面团

### 1.1 组成蛋白质

麸质并非单一蛋白，而是两大类蛋白相互作用后的产物：

| 蛋白质 | 英文名 | 分子特征 | 对面团的贡献 |
|--------|--------|---------|------------|
| **醇溶蛋白** | Gliadin | 单体，球形，富含 Gln/Pro，溶于70%乙醇 | 赋予**延展性**（viscosity/extensibility） |
| **谷蛋白** | Glutenin | 多聚体，线状，通过链间二硫键交联 | 赋予**弹性**（elasticity/strength） |

> ⚙️ 物理类比：Gliadin 像润滑油，让面团顺滑可延展；Glutenin 像弹簧，让面团拉伸后能回弹。两者比例（通常约 1∶1）决定面粉品质。

### 1.2 麸质网络的形成过程

1. **干粉阶段**：Gliadin 和 Glutenin 以独立颗粒形式存在于胚乳中，互不接触。
2. **加水阶段**：蛋白质吸水溶胀，分子开始接触并展开。
3. **揉面阶段（机械作用）**：
   - 链内二硫键（–S–S–）断裂重组；
   - Glutenin 聚合体拉伸、排列；
   - Gliadin 插入 Glutenin 骨架的孔隙，形成**黏弹性网络**。
4. **发酵阶段**：酵母释放 CO₂，气泡被麸质网捕获并稳定，面团膨胀。
5. **烘烤阶段（>75°C）**：蛋白质变性定型，淀粉糊化，麸质网固化为最终结构。

### 1.3 高筋 vs 低筋：蛋白质含量决定一切

| 面粉类型 | 蛋白质含量 | 麸质强度 | 适用产品 |
|---------|-----------|---------|---------|
| 高筋粉（面包粉） | 12–14% | 强，韧性高 | 面包、贝果、比萨 |
| 中筋粉（通用粉） | 10–12% | 中等 | 馒头、面条、饺子皮 |
| 低筋粉（糕点粉） | 7–9% | 弱，柔软 | 蛋糕、饼干、可颂 |
| 无麸质替代粉 | — | 无 | 专供麸质敏感者 |

## 二、麸质相关疾病谱——三类截然不同的机制

麸质相关疾病可分为三大类，机制各异，不可混淆：

```
小麦摄入后出现症状
        │
        ├─► 有特异性抗体（tTG-IgA等）＋ 肠绒毛萎缩？
        │       YES → 乳糜泻（Celiac Disease）[自身免疫]
        │
        ├─► 无 tTG-IgA，但有小麦特异性 IgE？
        │       YES → 小麦过敏（Wheat Allergy）[IgE 介导过敏]
        │
        └─► 两者均阴性，但症状实际存在？
                YES → 非乳糜泻麸质敏感（NCGS）[机制未明]
```

## 三、乳糜泻（Celiac Disease）——最严重的麸质相关疾病

### 3.1 基本情况

- **全球患病率**：约 1%，但仅 30% 得到确诊（大量误诊为 IBS 等）
- **发病机制**：基因 × 环境 × 麸质触发的**自身免疫性疾病**
- **靶器官**：小肠黏膜（尤其是十二指肠和空肠近段）

### 3.2 分子发病机制（详解）

```
① 麸质摄入
   ↓
② 肠道渗透性增加（可能由感染、压力等诱因引发）
   ↓
③ Gliadin 肽段进入固有层，被组织转谷氨酰胺酶（tTG2）修饰
   脱氨基作用：Gln → Glu，增加与 HLA 分子的亲和力
   ↓
④ 抗原呈递细胞（DC）展示修饰肽段，通过 HLA-DQ2/DQ8
   激活 CD4+ T 细胞（Th1 和 Th17 路径）
   ↓
⑤ 炎症因子风暴：IFN-γ、TNF-α、IL-21 大量释放
   ↓
⑥ 上皮内淋巴细胞（IEL）增多，杀伤肠上皮细胞
   ↓
⑦ 肠绒毛萎缩 → 吸收面积减少 80–90%
   ↓
⑧ 营养吸收障碍（铁、钙、叶酸、脂溶性维生素等）
```

> 🧬 **遗传因素**：HLA-DQ2（~90% 患者）或 HLA-DQ8（~8% 患者）是必要不充分条件——携带者中仅 2–3% 最终发病，说明环境因素同样关键。

### 3.3 症状与表现

乳糜泻症状高度多样，分为**经典型**和**非经典型**：

**经典型（消化道症状）**
- 慢性腹泻（脂肪泻、恶臭粪便）
- 腹胀、腹痛
- 体重下降、发育迟缓（儿童）

**非经典型（肠外症状，常被忽视）**
- 缺铁性贫血（对补铁治疗无反应）
- 骨质疏松（钙 & 维生素 D 吸收障碍）
- 疱疹样皮炎（Dermatitis Herpetiformis，皮肤型乳糜泻）
- 麸质共济失调（小脑损伤，步态异常）
- 不孕不育、反复流产
- 周围神经病变、抑郁

### 3.4 诊断流程

```
血清学筛查（首选）
  └─► tTG-IgA 抗体 + 总 IgA（排除 IgA 缺乏）
      │
      阳性 ──► 内镜 + 十二指肠活检
               Marsh 分级 II–III → 确诊
      │
      可疑/阴性 ──► HLA-DQ2/DQ8 检测
                   阴性 → 基本排除乳糜泻
```

> ⚠️ **重要**：必须在**仍进食麸质**的情况下进行检测，否则抗体可能转阴导致漏诊。

### 3.5 治疗

**目前唯一有效的治疗方法：严格终身无麸质饮食（GFD）**

- 成效：绝大多数患者 6–12 个月后肠道黏膜修复，症状消失
- 严格程度：即使 < 10 mg/天的麸质摄入（约一粒面包屑）也可维持黏膜损伤
- 合并营养支持：补充铁、叶酸、维生素 D、钙、B12 等

## 四、非乳糜泻麸质敏感（NCGS）——最有争议的类别

### 4.1 定义与现状

NCGS 是指在排除乳糜泻和小麦过敏的前提下，摄入麸质后出现胃肠道或肠外症状，去除麸质后症状缓解的状态。

- 预估患病率：0.5–6%（缺乏客观诊断标志物，数字可信度有限）
- **尚无被公认的血清/组织学生物标志物**

### 4.2 "真麸质效应"还是安慰剂/FODMAPs？

2011 年 Biesiekierski 等的双盲交叉试验曾"证实" NCGS 存在，但同一研究组 2013 年的再验证研究却发现：
- 严格控制 FODMAPs（可发酵糖类）后，麸质与安慰剂**无显著差异**；
- 症状改善主要来自 FODMAPs 减少，而非麸质本身。

> 🔍 **当前共识**：部分 NCGS 患者可能实际上是对**小麦 FODMAPs（主要是果聚糖 fructans）**敏感，而非麸质蛋白本身。真正的麸质特异性敏感是否存在仍有争议。

## 五、小麦过敏——IgE 介导的速发型反应

| 特征 | 说明 |
|------|------|
| **机制** | 小麦蛋白特异性 IgE 介导的 I 型超敏反应 |
| **症状** | 荨麻疹、血管神经性水肿、哮喘、过敏性休克（数分钟至数小时内） |
| **特殊类型** | 小麦依赖运动诱发过敏性休克（WDEIA）：进食小麦后运动才触发 |
| **诊断** | 皮肤点刺试验 + 特异性 IgE 检测 |
| **治疗** | 避免小麦（不必要完全无麸质，燕麦等通常安全） |

## 六、无麸质饮食（GFD）——科学证据与误区

### 6.1 谁真正需要无麸质饮食？

| 人群 | 是否需要严格 GFD |
|------|----------------|
| 乳糜泻患者 | ✅ 必须，终身 |
| 疱疹样皮炎 | ✅ 必须 |
| 麸质共济失调 | ✅ 应尽早开始 |
| 小麦过敏 | ⚠️ 需避免小麦，但不必全无麸质 |
| NCGS | ⚠️ 可试验性限制，但需低 FODMAP 鉴别 |
| 无相关诊断的普通人 | ❌ 无明确科学益处 |

### 6.2 GFD 对普通人的风险

2023–2024 年的系统综述（汇总 27 项研究）表明：

- **体重/BMI**：无麸质饮食对无乳糜泻人群的体重无显著影响
- **营养风险**：长期 GFD 常导致 B 族维生素（叶酸、B₁₂、烟酸）、铁、钙、膳食纤维摄入不足
- **加工食品陷阱**：商业无麸质产品为弥补口感，往往添加更多糖、脂肪和添加剂

> 💡 若有人在"戒麸质"后感觉更好，原因往往是：减少了精制碳水和超加工食品，而非麸质本身的去除。

## 七、麸质在食品工业中的应用

### 7.1 核心烘焙功能

| 功能 | 原理 | 应用 |
|------|------|------|
| 保气性 | 麸质网捕获 CO₂，形成均匀气孔 | 面包、馒头蓬松结构 |
| 弹韧性 | Glutenin 的弹性回复力 | 面条 Q 弹口感、面包嚼劲 |
| 可塑性 | Gliadin 的流动性 | 薄饼、可颂层状结构 |
| 持水性 | 蛋白质网络锁住水分 | 延缓老化，延长保质期 |

### 7.2 面筋（活性小麦麸质）的工业用途

活性小麦麸质粉（Vital Wheat Gluten）是从小麦粉中分离的高纯度麸质，用途广泛：

- **面包改良剂**：提升低蛋白面粉的面团强度
- **素肉/烤麸/面筋**：高蛋白仿肉食材（蛋白含量 75–80%）
- **宠物食品**：增加干粮蛋白含量（三聚氰胺丑闻即因此掺假检测）
- **肉制品黏结剂**：火腿、香肠中改善持水性和切片性

### 7.3 "面筋"与"麸质"的中文语境辨析

| 术语 | 英文对应 | 说明 |
|------|---------|------|
| 麸质 | Gluten | 泛指面粉中的蛋白网络，医学/营养语境常用 |
| 面筋 | Gluten / Seitan | 从面团中洗出的固态麸质，烹饪食材 |
| 活性麸质 | Vital Wheat Gluten | 工业干粉，可重新水化恢复黏弹性 |

## 八、麸质与肠道健康——新兴研究

### 8.1 麸质对肠道通透性的作用

即使在非乳糜泻人群中，麸质也可能通过**连蛋白（Zonulin）**通路：
1. Gliadin 结合肠上皮 CXCR3 受体
2. 触发 Zonulin 释放
3. 紧密连接蛋白（Occludin/Claudin）暂时开放
4. 肠道通透性一过性升高

> 争议：这种效应在健康人群中是否具有临床意义尚不确定。

### 8.2 肠道菌群与麸质

- 乳糜泻患者的肠道菌群与健康人显著不同（厚壁菌门减少，拟杆菌门增加）
- 部分研究提示肠道菌群失调可能影响麸质肽段的代谢，加剧或保护性影响乳糜泻发病
- **益生菌是否可辅助治疗乳糜泻**：目前证据不足，尚处研究阶段

## 九、含麸质与无麸质食物速查表

### 含麸质谷物（需严格回避）

- 小麦（包括古代品种：斯佩尔特小麦、卡姆特、杜兰小麦）
- 大麦（啤酒、麦芽）
- 黑麦
- 普通燕麦（交叉污染风险，乳糜泻患者需选认证无麸质燕麦）

### 天然无麸质食物

- 大米、玉米、小米、高粱、荞麦、藜麦、苋籽
- 所有新鲜肉类、鱼类、蛋类
- 所有天然蔬菜、水果
- 豆类（大豆、鹰嘴豆等）
- 坚果、种子
- 纯乳制品

### 隐性麸质来源（高风险交叉污染或添加）

- 酱油（传统酿造含小麦）
- 啤酒、麦芽醋
- 方便食品的增稠剂、调味料
- 素肉、烤麸
- 部分燕麦片（生产线污染）

## 十、行动清单：如何判断自己是否应该减少麸质？

- [ ] **步骤 1**：如有以下症状，首先就医排查乳糜泻：慢性腹泻、不明原因贫血、骨质疏松（年轻人）、不孕不育
- [ ] **步骤 2**：检查前**不要**提前戒麸质，否则影响血清学结果
- [ ] **步骤 3**：血清学筛查：tTG-IgA + 总 IgA；阳性者做肠镜活检
- [ ] **步骤 4**：排除乳糜泻和小麦过敏后，若仍怀疑麸质问题，先尝试**低 FODMAPs 饮食**，区分是麸质还是果聚糖
- [ ] **步骤 5**：无上述诊断的健康人——无需无麸质饮食，保持均衡饮食更重要

## 核心结论

| 维度 | 关键认知 |
|------|---------|
| **化学本质** | 两类蛋白（Gliadin + Glutenin）遇水形成的黏弹性网络 |
| **食品功能** | 赋予面团弹性与保气性，是烘焙的灵魂 |
| **乳糜泻** | 严重自身免疫疾病，唯一治疗是严格终身 GFD |
| **NCGS** | 机制未明，可能大量混入 FODMAPs 敏感者 |
| **普通人** | 无需 GFD，反而需警惕 GFD 带来的营养失衡风险 |
| **商业 GFD 热潮** | 大多缺乏科学依据，减少的是精制碳水而非麸质本身的效益 |
