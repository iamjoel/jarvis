# Frontend
## UI
* [Base UI](https://base-ui.com/) An unstyled UI component library for building accessible user interfaces. From the creators of Radix, Floating UI, and Material UI. [Shadcn/ui](https://ui.shadcn.com/) is also based on it(Prev based on Radix UI). 


## StateManagement
* [Zustand](https://zustand-demo.pmnd.rs/) A small, fast, and scalable bearbones state management solution.
* [Jotai](https://jotai.org/) Jotai takes an atomic approach to global React state management.

## DataFetching
* [Tanstack Query](https://tanstack.com/query/latest) Powerful asynchronous state management, server-state utilities and data fetching.
* [oRPC](https://orpc.dev/) Easy to build APIs that are end-to-end type-safe and adhere to OpenAPI standards.

## Framework
* [Next.js](https://nextjs.org/) The React Framework for Production.

## Presentation
* [remark-directive](https://github.com/remarkjs/remark-directive) parse directives in markdown. Can be used to create custom syntax for markdown, and then transform it to HTML or React components.
