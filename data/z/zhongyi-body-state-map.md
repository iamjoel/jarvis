# 中医身体状态描述图谱

## 高密度摘要

- **一句话结论**：中医的身体状态词不是单一疾病名，而是一套从病因、病位、寒热、虚实、气血津液、脏腑功能和体质倾向来描述身体失衡的语言系统。
- **核心机制**：先用八纲判断大方向，再看六淫或内生因素的性质，接着落到气血津液与脏腑功能，最后区分当下证候和长期体质。
- **判断入口**：遇到“湿、寒、热、虚、瘀、痰、郁、火、燥”等词，先问它属于病因、证候、病位、体质还是症状风格。
- **常见误区**：不要把中医状态词直接翻译成现代医学诊断；“湿热”“痰湿”“气虚”“血瘀”都是模式语言，需要结合症状组合、舌脉、病程和危险信号判断。
- **相关文档**：[中医里的“湿”与身体状态描述框架](./zhongyi-shi-and-body-state-descriptors.md)、[健康索引](../health.md)、[生物医学证据判断](../b/biomedical-evidence-literacy.md)。

## 先给结论

中医里的“身体状态”大致不是在回答“你得了哪一种现代疾病”，而是在回答：

1. 身体失衡偏外还是偏内？
2. 偏寒还是偏热？
3. 偏虚还是偏实？
4. 是气、血、津液哪一类运行出问题？
5. 主要影响哪个功能系统？
6. 是短期证候，还是长期体质倾向？

所以“湿”“寒”“热”“虚”“瘀”“痰”“郁”这些词，最好理解为传统医学里的状态标签。它们描述的是症状组合的方向和性质，而不是现代医学里的单个病名、单个器官或单个化验指标。

## 一张总图

| 层级 | 代表概念 | 它回答的问题 |
|------|------|------|
| 八纲 | 阴阳、表里、寒热、虚实 | 大方向是什么 |
| 六淫 | 风、寒、暑、湿、燥、火 | 外来或环境性因素呈现什么性质 |
| 内生状态 | 内风、内寒、内湿、内燥、内火 | 失衡从身体内部怎样生成 |
| 气血津液 | 气虚、气滞、血虚、血瘀、津亏、痰饮、水湿 | 运行材料和动力系统哪里不顺 |
| 脏腑证候 | 脾虚、肾虚、肝郁、肺失宣降、胃热等 | 哪个功能系统牵涉最多 |
| 体质 | 平和、气虚、阳虚、阴虚、痰湿、湿热、血瘀、气郁、特禀 | 长期倾向是什么 |

实际辨证通常不是只用一个词，而是组合表达，比如“脾虚夹湿”“肝郁气滞”“湿热下注”“气滞血瘀”“阴虚火旺”。

## 一、八纲：所有状态词的总开关

八纲是中医辨证的第一层框架。

| 维度 | 含义 | 常见线索 |
|------|------|------|
| 表 / 里 | 病位深浅 | 表偏外感、肌表；里偏脏腑、气血津液 |
| 寒 / 热 | 反应方向 | 寒偏怕冷、清稀、冷痛；热偏口渴、烦躁、红黄、灼热 |
| 虚 / 实 | 正邪关系 | 虚偏不足、疲乏、久病；实偏壅滞、胀满、疼痛、有余 |
| 阴 / 阳 | 总体归类 | 阴偏寒、静、内、下、虚；阳偏热、动、外、上、实 |

八纲的重要性在于：同一个“湿”，可以是寒湿，也可以是湿热；同一个“痛”，可以是寒凝、气滞、血瘀，也可以是虚痛。方向不同，解释和处理就不同。

## 二、六淫：风寒暑湿燥火

六淫原本是外感病因语言，也可以用来描述症状呈现出来的性质。

| 名称 | 核心意象 | 常见身体状态 |
|------|------|------|
| 风 | 变化快、游走、开泄、动 | 头痛、怕风、瘙痒、抽动、症状位置变化 |
| 寒 | 收缩、凝滞、冷、痛 | 怕冷、冷痛、清稀、拘急、遇冷加重 |
| 暑 | 夏季炎热、升散、耗气伤津 | 发热、汗多、口渴、乏力、胸闷 |
| 湿 | 沉重、黏滞、浑浊、趋下 | 身重、困倦、苔腻、大便黏、分泌物浊 |
| 燥 | 干涩、伤津 | 口干、咽干、鼻干、皮肤干、大便干 |
| 火 / 热 | 炎上、扰动、消耗 | 发热、口渴、烦躁、红肿热痛、尿黄 |

六淫不是气象词的简单搬运。例如“湿”不只是环境潮湿，也可指身体内部出现黏滞、重浊、运化不畅的状态；“风”也不只是刮风，还可用于描述变化快、游走、抽动、瘙痒等特征。

## 三、内生状态：从身体内部生成的风寒湿燥火

中医也会说“内风”“内寒”“内湿”“内燥”“内火”。这类词强调失衡不是外界直接造成，而是身体内部功能失调后形成。

| 名称 | 直观理解 | 常见关联 |
|------|------|------|
| 内风 | 内部失衡导致“动摇、抽动、眩晕” | 肝风、眩晕、震颤、抽搐类描述 |
| 内寒 | 阳气不足或寒从内生 | 怕冷、腹冷、便溏、四肢不温 |
| 内湿 | 水液运化失常 | 困重、苔腻、腹胀、痰多、浮肿 |
| 内燥 | 津液不足或濡养不足 | 口干、皮肤干、大便干 |
| 内火 | 内部热盛或虚热上扰 | 口疮、烦躁、口苦、失眠、咽痛 |

这一层常和脏腑功能相连，例如“脾虚生湿”“阴虚内热”“肝郁化火”。

## 四、气血津液：身体运行材料的状态

气、血、津液是中医解释身体功能的三类核心材料和动力。常见状态如下：

| 名称 | 大意 | 常见线索 |
|------|------|------|
| 气虚 | 推动、温煦、防御不足 | 乏力、气短、懒言、自汗、容易疲劳 |
| 气滞 | 气机运行不畅 | 胀、闷、痛无定处、叹气、情绪郁结 |
| 气逆 | 气机上逆 | 咳喘、呃逆、恶心、嗳气 |
| 血虚 | 血的濡养不足 | 面色淡、头晕、心悸、眼干、肢麻 |
| 血瘀 | 血行不畅 | 刺痛、痛处固定、舌暗、瘀点、肌肤甲错 |
| 血热 | 血分有热 | 出血、皮疹、烦热、舌红 |
| 津亏 | 津液不足 | 口干、咽干、皮肤干、大便干 |
| 水湿 | 水液停聚偏重浊 | 身重、浮肿、苔腻、困倦 |
| 痰饮 | 水湿凝聚成更黏滞的状态 | 痰多、胸闷、眩晕、结块类解释常会用到 |

这一层非常适合用来理解“为什么状态词会组合”：气滞久了可能影响血行，形成气滞血瘀；水湿久留可能成痰；津亏可生燥；阴虚可生内热。

## 五、常见组合词

中医状态词最常见的用法是组合，而不是孤立出现。

| 组合 | 直观理解 | 常见线索 |
|------|------|------|
| 寒湿 | 寒 + 湿 | 沉重、怕冷、清稀、关节冷痛、遇冷加重 |
| 湿热 | 湿 + 热 | 油腻、口苦、苔黄腻、分泌物黄浊、尿黄 |
| 痰湿 | 湿久成痰，黏滞重浊 | 体胖、胸闷、多痰、口黏、苔腻 |
| 气滞 | 运行受阻 | 胀闷、疼痛位置变化、情绪相关 |
| 气滞血瘀 | 气不行则血不畅 | 胀痛转刺痛、痛处固定、舌暗 |
| 阴虚火旺 | 阴液不足，虚热上扰 | 口干、盗汗、五心烦热、失眠 |
| 阳虚水泛 | 温煦不足，水液停聚 | 怕冷、浮肿、尿少、疲乏 |
| 肝郁脾虚 | 情志疏泄不畅影响运化 | 情绪郁闷、腹胀、便溏、食欲差 |
| 脾虚夹湿 | 运化不足导致湿停 | 乏力、腹胀、困重、大便黏或溏 |
| 食积 | 饮食停滞 | 腹胀、嗳腐、口气、食欲下降 |

这些组合词通常比单字更有判断价值。“热”和“湿热”，“虚”和“阳虚”，“瘀”和“气滞血瘀”，指向并不一样。

## 六、脏腑证候：状态落到功能系统

中医的脏腑更多是功能系统，不等同于现代解剖器官。

| 功能系统 | 常见状态词 | 大意 |
|------|------|------|
| 脾胃 | 脾虚、胃寒、胃热、湿阻中焦、食积 | 消化、吸收、转输、水湿运化相关 |
| 肝 | 肝郁、肝火、肝血虚、肝风 | 情志疏泄、气机调达、筋目相关 |
| 心 | 心气虚、心血虚、心火、心神不宁 | 心悸、睡眠、精神活动相关 |
| 肺 | 肺气虚、肺热、肺燥、痰湿阻肺 | 呼吸、皮毛、宣发肃降相关 |
| 肾 | 肾阳虚、肾阴虚、肾气不固、水液失调 | 生长衰老、温煦、封藏、水液调节相关 |

例如“湿”如果落在脾胃，可能表现为腹胀、食欲差、大便黏滞；如果落在下焦，可能表现为尿黄、带下、下肢困重；如果和肺相关，可能被解释为痰湿阻肺。

## 七、体质：长期倾向，不等于当下疾病

《中医体质分类与判定》把体质分为 9 种基本类型。2025 年发布的 GB/T 46939-2025 国家标准规定了 9 种基本体质类型的特征和判定，2026 年 4 月 1 日起实施。

| 体质 | 核心倾向 |
|------|------|
| 平和质 | 阴阳气血调和，状态相对稳定 |
| 气虚质 | 容易疲乏、气短、自汗、抗压能力弱 |
| 阳虚质 | 怕冷、手脚凉、温煦不足 |
| 阴虚质 | 偏干、偏热、津液不足 |
| 痰湿质 | 黏滞重浊、水湿内停、体胖或腹部松软倾向 |
| 湿热质 | 湿与热并见，油腻、口苦、苔黄腻倾向 |
| 血瘀质 | 血行不畅，肤色晦暗、固定疼痛倾向 |
| 气郁质 | 气机郁滞，情绪不舒、胸闷叹气倾向 |
| 特禀质 | 过敏或先天禀赋特殊倾向 |

体质是长期倾向，证候是当下状态。一个人可以是痰湿质，但当下未必有明显湿证；也可以不是痰湿质，却因饮食、环境、睡眠和运动变化短期出现湿阻。

## 八、怎么读懂一个中医状态词？

可以按这 5 个问题拆：

1. **它属于哪一层？**  
   “风寒暑湿燥火”偏病因和性质；“寒热虚实”偏总方向；“气虚、血瘀、痰湿”偏气血津液；“痰湿质、湿热质”偏体质。

2. **它是单词还是组合词？**  
   单独的“湿”信息量较低，“寒湿”“湿热”“痰湿”“脾虚夹湿”才更接近临床判断。

3. **它描述的是症状风格还是病位？**  
   “湿”描述风格，“下焦湿热”才说明病位倾向，“脾虚夹湿”说明功能系统。

4. **它是虚还是实？寒还是热？**  
   这是避免误判的关键。虚寒和实热、寒湿和湿热，处理方向不同。

5. **有没有现代医学危险信号？**  
   持续发热、胸痛胸闷、呼吸困难、黄疸、明显水肿、黑便便血、体重异常下降、持续腹泻、剧烈疼痛、神经功能异常等，应优先就医排查。

## 和现代医学的关系

中医状态词可以帮助组织体感和症状模式，但不能直接等同现代医学诊断。

比较稳妥的使用方式是：

- 用中医词理解身体状态的“方向”：寒热、虚实、燥湿、气血运行。
- 用现代医学排查明确疾病和危险信号：感染、炎症、内分泌、心血管、肝肾功能、消化系统、免疫系统等。
- 不把“湿气重”“肝火旺”“气血虚”当成万能解释。
- 不用单个状态词替代检查、诊断和治疗决策。
