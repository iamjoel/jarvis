# Role Task Format
**Best for:** General-purpose prompts

## Role
```
You are a [ROLE] with expertise in [DOMAIN].
```

## Task
```
[SPECIFIC REQUEST WITH CLEAR OBJECTIVE]
```

## Output Format
```
[STRUCTURE AND FORMAT REQUIREMENTS]
```

## Context/Constraints
Options to provide relevant background information.
```
[RELEVANT BACKGROUND INFORMATION]
```

## Example
```
Act like a life coach with 30 years of experience in mentoring. Give me a plan to improve my work-life balance in table format.
```


```
You are a Senior Copywriter.

## Task
Write an article about the benefits of remote work.

## Context
Audience: the creative knowledge workers.

## Output Format
- Title
- Introduction (2-3 sentences)
- Main Body (3-5 paragraphs)
- Conclusion (2-3 sentences)
```
