# 皮质醇

## 高密度摘要

- **一句话结论**：皮质醇是反映 HPA 轴、昼夜节律和肾上腺功能的关键糖皮质激素；长期过高会增加高血压、高血糖、骨质疏松、感染和肌肉流失风险，过低则可能提示肾上腺功能不全甚至肾上腺危象。
- **核心机制**：下丘脑-垂体-肾上腺轴通过 CRH、ACTH 和皮质醇负反馈调节能量动员、血压、血糖、炎症、免疫和应激反应；正常皮质醇早晨高、夜间低。
- **判断入口**：不要因为“压力大”就自行测随机皮质醇；出现库欣综合征特征、肾上腺功能不全症状、长期使用糖皮质激素、垂体/肾上腺疾病或医生要求评估时才重点检测。
- **常见误区**：随机血皮质醇不能筛查库欣综合征；单次低值不能直接诊断肾上腺功能不全；“降低皮质醇”不是越低越好，真正目标是恢复正常节律和病因管理。

## 一句话介绍

皮质醇（cortisol）是肾上腺皮质分泌的主要糖皮质激素，用来维持血压、血糖、能量代谢、抗炎反应和身体对压力的适应能力。

## 它在身体里做什么

皮质醇不是单纯的“压力坏激素”，而是人体维持清醒、能量和生存反应的基础激素。

| 作用 | 具体意义 |
|------|------|
| 血糖与能量 | 促进糖异生，帮助身体在饥饿、运动、感染和心理压力下维持可用能量 |
| 血压与循环 | 帮助血管维持对儿茶酚胺的反应，支持血压稳定 |
| 炎症与免疫 | 抑制过度炎症反应；长期过高会压制免疫、防御和组织修复 |
| 昼夜节律 | 清晨升高帮助醒来，白天下降，夜间维持低水平以利于睡眠 |
| 应激适应 | 在急性压力下动员资源；长期压力、睡眠不足和疾病可扰乱节律 |

## 异常值有什么问题

### 长期过高

真正需要警惕的是长期暴露在过高皮质醇下，而不是一次紧张、熬夜或运动后的短暂升高。

常见后果包括：

- 腹部和面部脂肪增加、四肢相对变细、肌肉无力。
- 血压升高、血糖升高、胰岛素抵抗、2 型糖尿病风险上升。
- 骨量下降、骨质疏松和骨折风险增加。
- 皮肤变薄、容易淤青、紫纹、伤口愈合变慢。
- 感染风险增加，情绪低落、焦虑、记忆和注意力下降。

长期明显过高需要考虑库欣综合征。常见原因包括长期使用糖皮质激素药物、垂体 ACTH 分泌过多、异位 ACTH 分泌、肾上腺肿瘤或肾上腺结节性增生。

### 长期过低

皮质醇不足会削弱身体维持血压、血糖和应激反应的能力。

常见表现包括：

- 长期疲劳、肌肉无力、食欲下降、体重下降。
- 腹痛、恶心、呕吐、腹泻。
- 低血压，尤其站立时头晕或晕厥。
- 低血糖、想吃咸食、情绪低落或易激惹。
- 原发性肾上腺功能不全时可能出现皮肤色素沉着。

如果出现严重乏力、低血压、意识模糊、持续呕吐腹泻、严重腹痛或疑似肾上腺危象，应按急症处理，而不是等待常规体检复查。

## 正常值的范围

皮质醇的“正常范围”强烈依赖采样时间、样本类型、检测方法、实验室参考区间、睡眠作息、疾病状态和用药背景。

| 指标 | 常见参考或判断方式 | 说明 |
|------|------|------|
| 早晨 8 点血清皮质醇 | 常见约 5-25 mcg/dL，约 138-690 nmol/L | 适合初步观察早晨峰值，但不能单独判断所有问题 |
| 下午血清皮质醇 | 通常低于早晨 | 有时用于观察昼夜下降趋势 |
| 午夜/晚间唾液皮质醇 | 正常应较低 | 常用于库欣综合征筛查，因为库欣常表现为夜间皮质醇压不下来 |
| 24 小时尿游离皮质醇 | 看全天游离皮质醇暴露量 | 常用于库欣综合征筛查，通常需要按要求完整留尿 |
| ACTH 兴奋试验 | ACTH 注射后皮质醇应升高 | 常用于确认肾上腺功能不全 |

几个实用阈值：

- 早晨 8 点血清皮质醇在 5-25 mcg/dL 之间常见于正常参考范围，但具体以实验室报告为准。
- Endocrine Society 指南把早晨皮质醇低于 5 mcg/dL 且结合 ACTH 作为肾上腺功能不全的初步提示，仍需在条件允许时用促肾上腺皮质激素兴奋试验确认。
- ACTH 兴奋试验中，30 或 60 分钟峰值皮质醇低于 18 mcg/dL（500 nmol/L，且受检测方法影响）提示肾上腺功能不全。

## 关联指标

| 指标 | 补充判断什么 |
|------|------|
| ACTH | 区分问题更偏垂体/下丘脑信号不足，还是肾上腺本身反应不足 |
| DHEA-S | 反映肾上腺雄激素分泌，可辅助理解肾上腺皮质功能 |
| 醛固酮、肾素 | 判断是否合并盐水调节异常，尤其原发性肾上腺功能不全 |
| 钠、钾 | 肾上腺功能不全可出现低钠、高钾；也帮助判断危急程度 |
| 空腹血糖、HbA1c | 皮质醇过高会推高血糖和胰岛素抵抗风险 |
| 血压 | 皮质醇过高常伴高血压；过低或危象时可低血压 |
| 骨密度 | 长期高皮质醇或长期糖皮质激素用药会增加骨量流失 |
| HRV、静息心率、睡眠 | 反映压力和恢复趋势，但不能替代皮质醇检测 |

## 如何检测

### 常见检测方式

| 检测 | 适合场景 | 注意事项 | 常见费用 |
|------|------|------|------|
| 血清皮质醇 | 初步评估早晨峰值、肾上腺功能不足线索、部分动态试验 | 通常按医生要求在早晨抽血；前一天可能需要避免剧烈运动；药物影响很大 | 常见约 50-200 元人民币 |
| 唾液皮质醇 | 观察晚间/午夜皮质醇，筛查库欣综合征 | 采样前通常避免进食、饮水、刷牙、牙线等；按试剂盒要求采集 | 常见约 100-400 元人民币 |
| 24 小时尿游离皮质醇 | 估计全天游离皮质醇暴露，筛查库欣综合征 | 需要完整收集 24 小时尿液，漏收会影响结果 | 常见约 100-400 元人民币 |
| 地塞米松抑制试验 | 筛查皮质醇是否能被正常反馈抑制 | 需按医嘱服药和次日抽血；部分药物会干扰结果 | 常见约 100-500 元人民币 |
| ACTH 兴奋试验 | 确认肾上腺是否能对 ACTH 作出反应 | 属于动态内分泌试验，应由医疗机构执行 | 常见约 300-1000 元人民币 |

价格会因城市、医院级别、医保、检测方法和是否包含 ACTH/药物/动态试验而差异很大。

### 什么时候不建议自己随便测

- 只是觉得最近压力大、想看“皮质醇是不是高”。
- 想用一次随机血皮质醇判断库欣综合征。
- 想靠居家检测决定是否停用、减量或加用糖皮质激素。
- 正在使用口服、注射、吸入、外用或关节腔注射糖皮质激素，却没有把用药史告诉医生。

糖皮质激素药物、雌激素、抗癫痫药、雄激素、急性疾病、夜班作息、睡眠不足、剧烈运动、怀孕和严重压力都可能影响结果解释。

## 如何恢复正常

皮质醇异常的处理逻辑不是“越低越好”，而是先判断它是生理波动、节律紊乱、药物影响，还是明确内分泌疾病。

### 生活方式层面

适合压力大、睡眠差、恢复差，但没有库欣或肾上腺功能不全红旗信号的人：

- 固定起床时间，早晨接触自然光，夜间减少强光和屏幕刺激。
- 保证睡眠时长和规律性，避免长期熬夜和昼夜节律漂移。
- 规律运动，以中等强度有氧和力量训练为主；避免长期过度训练和恢复不足。
- 用呼吸练习、冥想、瑜伽、太极、散步、户外活动降低持续交感激活。
- 控制过量咖啡因、酒精、含糖饮料和高糖零食；饮食以高营养密度、足量蛋白、蔬果、全谷物和健康脂肪为主。
- 把“降低皮质醇”改成“让早晨该高时高、夜间该低时低”，这比追求单个低值更合理。

### 需要医生评估的情况

以下情况更适合内分泌科评估：

- 圆脸、肩背脂肪堆积、腹型肥胖但四肢变细、紫纹、容易淤青、肌无力、高血压、高血糖同时进展。
- 不明原因长期乏力、体重下降、低血压、站立头晕、低钠高钾、皮肤变黑、反复低血糖。
- 长期或大剂量使用糖皮质激素，或近期停用后出现明显虚弱、低血压、恶心呕吐。
- 体检发现肾上腺意外瘤、垂体病变，或医生怀疑库欣综合征/肾上腺功能不全。

### 药物和疾病层面

- 库欣综合征需要先区分外源性糖皮质激素、垂体、异位 ACTH 和肾上腺来源；治疗可能涉及调整药物、手术、放疗或抑制皮质醇生成的药物。
- 肾上腺功能不全需要明确原发性、继发性或三发性原因；确诊后通常需要糖皮质激素替代，原发性患者还可能需要盐皮质激素替代。
- 已经长期使用糖皮质激素的人不能自行突然停药；需要由医生根据疾病、剂量、疗程和 HPA 轴恢复情况逐步调整。
