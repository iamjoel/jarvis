# 咳嗽深度指南

> **核心概念**：咳嗽是人体呼吸道最重要的保护性反射，用于清除气道内的分泌物、异物和有害物质。它既可以是短暂的良性症状，也可以是严重疾病的早期信号。全球约 10–40% 的成年人有慢性咳嗽困扰，是就医最常见的主诉之一。正确识别咳嗽类型与病因，才能避免无效用药和误诊延误。

---

## 一、咳嗽的分类：按持续时间

| 类型 | 持续时间 | 最常见病因 |
|------|---------|-----------|
| **急性咳嗽** | < 3 周 | 病毒性上呼吸道感染（普通感冒）、急性支气管炎、肺炎、急性鼻窦炎 |
| **亚急性咳嗽** | 3–8 周 | 感染后气道高反应性、哮喘首次发作、百日咳 |
| **慢性咳嗽** | > 8 周（儿童 > 4 周） | 鼻后滴流（UACS）、哮喘、胃食管反流（GERD）、嗜酸性支气管炎、ACEI药物副作用 |

> ⚠️ **重要提醒**：咳嗽时间越长，需要系统排查的病因越多；超过 8 周的慢性咳嗽有时背后隐藏肺结核、肺癌等严重疾病，不可掉以轻心。

---

## 二、急性咳嗽的主要成因

### 2.1 病毒性上呼吸道感染（最常见）

- **占比**：急性咳嗽中约 70–80%
- **机制**：病毒（鼻病毒、流感病毒、冠状病毒等）感染鼻咽、气管黏膜，激活咳嗽受体；黏液分泌增多刺激气道
- **特征**：常伴流涕、咽痛、低热；咳嗽多为干咳或少量白色黏痰；通常 1–2 周自愈
- **误区**：普通感冒所致咳嗽**不需要**抗生素，使用抗生素无效且有害

### 2.2 急性细菌性支气管炎 / 肺炎

- **区别于感冒**：持续高热（>38.5°C）、咳黄绿色脓痰、呼吸困难、胸痛
- **高危人群**：老年人、免疫力低下者、慢性肺病患者
- **诊断**：胸部 X 线可见浸润影；血常规白细胞、C 反应蛋白升高
- **处理**：需要抗生素治疗，根据病原体和药敏结果选药

### 2.3 急性鼻窦炎

- **机制**：鼻窦炎症引起大量分泌物向后流入咽喉，刺激咳嗽受体（鼻后滴流）
- **特征**：晨起或卧位时咳嗽加重，鼻腔有压迫感、面部疼痛
- **处理**：生理盐水冲洗鼻腔、鼻腔激素喷雾；细菌性鼻窦炎需抗生素

### 2.4 COVID-19 及其他病毒性肺炎

- **特点**：干咳为主，可伴嗅觉/味觉丧失、持续低热、乏力；重症出现呼吸困难
- **后遗症咳嗽**：部分患者感染 COVID-19 后出现持续数周至数月的咳嗽（"长新冠"）

## 三、慢性咳嗽的主要成因（重点）

慢性咳嗽病因复杂，成人中最常见的三大病因（"慢性咳嗽三联征"）覆盖约 90% 的病例：

### 3.1 上气道咳嗽综合征 / 鼻后滴流（UACS/PND）——最常见

- **机制**：鼻炎（过敏性或非过敏性）、鼻窦炎、鼻息肉导致鼻腔分泌物向后流入咽喉，刺激咽喉部咳嗽受体
- **特征**：
  - 咽部有痰感，需频繁清嗓
  - 咳嗽在清晨、卧位时加重
  - 常伴鼻塞、打喷嚏、流涕
- **诊断线索**：鼻咽部检查可见黏液或鹅卵石样改变
- **治疗**：
  - 过敏性鼻炎：鼻腔皮质激素（如布地奈德鼻喷）+ 第二代抗组胺药（如氯雷他定）
  - 非过敏性鼻炎：第一代抗组胺药（如苯海拉明）+ 减充血剂
  - 鼻窦炎：鼻腔盐水冲洗、抗生素（细菌性）、必要时手术

### 3.2 哮喘 / 咳嗽变异型哮喘（CVA）

- **咳嗽变异型哮喘**：哮喘的特殊类型，**唯一症状是慢性干咳**，无明显喘息，常被误诊为"感冒后遗症"
- **机制**：气道慢性炎症导致气道高反应性；冷空气、运动、尘螨、花粉等诱发咳嗽发作
- **特征**：
  - 夜间或凌晨咳嗽加重（典型特征）
  - 运动、冷空气、异味诱发
  - 季节性或过敏史
- **诊断**：肺功能（支气管舒张试验阳性）、峰流速变异率、诱导痰嗜酸性粒细胞
- **治疗**：吸入性糖皮质激素（ICS，如布地奈德）为一线；联合长效 β₂ 激动剂（LABA）；白三烯受体拮抗剂（孟鲁司特）作为辅助

### 3.3 胃食管反流性咳嗽（GERC）

- **机制**：
  1. 胃酸反流至咽喉，直接刺激咳嗽受体（微量误吸）
  2. 酸激活食管-支气管迷走神经反射，间接引起气道收缩
- **特征**：
  - 进餐后、平卧时咳嗽加重
  - 常伴烧心、反酸、嗳气（但部分患者无消化道症状！）
  - 餐后 1–2 小时内明显
- **诊断**：24 小时食管 pH-阻抗监测；质子泵抑制剂（PPI）诊断性治疗有效可确诊
- **治疗**：
  - 生活方式：减重、戒酒戒烟、少量多餐、餐后 2 小时内不卧位、床头抬高 15–20 cm
  - 药物：质子泵抑制剂（奥美拉唑、兰索拉唑）+ 促动力药（莫沙必利）；疗程 8–12 周
  - 注意：PPI 治疗 GERC 咳嗽起效慢，需要 2–3 个月才能判断效果

### 3.4 嗜酸性粒细胞性支气管炎（EB）

- **与哮喘的区别**：同样有气道嗜酸性粒细胞浸润，但**没有气道高反应性**，肺功能正常，无喘鸣
- **特征**：干咳为主，对吸入刺激物敏感
- **诊断**：诱导痰嗜酸性粒细胞 > 3%，支气管激发试验阴性
- **治疗**：吸入性糖皮质激素有效

### 3.5 ACEI 药物性咳嗽

- **原因**：ACE 抑制剂（如卡托普利、依那普利，常用于高血压/心衰）抑制缓激肽降解 → 缓激肽在气道积聚 → 刺激咳嗽感受器
- **特征**：
  - 干咳、刺激性咳嗽，无痰
  - 服药后数周至数月出现
  - 所有 ACEI 类药物均可引起，亚洲人（含中国人）发生率高达 30–40%，远高于西方人（5–20%）
- **处理**：停用 ACEI，换用 ARB（血管紧张素受体阻断剂，如缬沙坦）1–4 周后咳嗽消退

### 3.6 慢性阻塞性肺疾病（COPD）

- **背景**：长期吸烟（≥20 包年）最重要危险因素
- **特征**：慢性咳嗽（晨咳为著）+ 咳痰 + 进行性呼吸困难；冬季急性加重
- **诊断**：肺功能（FEV₁/FVC < 70%）
- **治疗**：戒烟（最重要！）、支气管扩张剂（LABA、LAMA）、吸入激素（重度用）、肺康复

### 3.7 其他需排除的重要病因

| 病因 | 预警信号 | 处理 |
|------|---------|------|
| **肺结核** | 咳嗽 > 2 周 + 低热、盗汗、体重下降；高危人群（免疫低下、密切接触史） | 结核菌素试验、痰涂片/培养、胸部CT；转结核专科 |
| **肺癌** | 年龄 > 40 岁 + 大量吸烟史；痰中带血、声音嘶哑、体重骤降 | 胸部 CT（低剂量）、支气管镜；肿瘤科转诊 |
| **支气管扩张** | 每日大量脓痰、反复肺部感染、咳血 | 胸部高分辨 CT；感染控制、体位引流 |
| **异物吸入** | 儿童多见；突发剧烈呛咳，有明确吸入史 | 立即支气管镜取出 |
| **心功能不全** | 夜间阵发性呼吸困难、端坐呼吸、双下肢水肿 | 心脏评估、利尿剂治疗 |
| **百日咳** | 阵发性痉挛性咳嗽 + 吸气性鸡鸣声（成人可不典型） | 鼻咽拭子 PCR；阿奇霉素 |

## 四、诊断思路：系统排查流程

```
咳嗽
│
├─ 有红旗症状？（咳血、体重骤降、吸烟史+中老年、持续发热、呼吸困难）
│   └─ 是 → 胸部CT + 专科转诊，优先排除肿瘤/结核/肺炎
│
├─ 急性（< 3 周）
│   └─ 多为病毒感染 → 对症处理；有肺炎征象→胸片+血常规
│
├─ 亚急性（3–8 周）
│   └─ 多为感染后残留 → 观察或短期止咳；不缓解→按慢性流程评估
│
└─ 慢性（> 8 周）
    ├─ 正在服用 ACEI？ → 停药观察 4 周
    ├─ 大量吸烟史？ → 肺功能排查 COPD
    ├─ 鼻塞/清嗓/晨起咳嗽？ → 试验性治疗 UACS（鼻激素+抗组胺）
    ├─ 夜间/运动诱发干咳？ → 试验性治疗哮喘（ICS 4–8 周）
    ├─ 餐后/平卧加重？ → 试验性治疗 GERD（PPI 8–12 周）
    └─ 以上均无效 → 诱导痰、支气管镜、进一步影像
```

## 五、治疗方案总表

### 5.1 对因治疗（优先）

| 病因 | 一线治疗 | 二线 / 补充 |
|------|---------|-----------|
| 病毒性感冒 | 对症（退热、祛痰）、充足水分、休息 | 蜂蜜（> 1岁儿童可用）；避免抗生素 |
| 细菌性肺炎 | 抗生素（阿莫西林/克拉维酸、阿奇霉素等，按病情选择） | 祛痰剂辅助 |
| UACS（过敏性） | 鼻腔皮质激素 + 第二代抗组胺药（氯雷他定/西替利嗪） | 白三烯受体拮抗剂（孟鲁司特） |
| UACS（非过敏性） | 第一代抗组胺药（苯海拉明）+ 盐酸伪麻黄碱 | 鼻腔皮质激素 |
| 哮喘 / CVA | 吸入性糖皮质激素（布地奈德、氟替卡松）± LABA | 孟鲁司特；严重时口服激素短期冲击 |
| 嗜酸性支气管炎 | 吸入性糖皮质激素（同哮喘） | — |
| GERC | PPI（奥美拉唑 20 mg bid）+ 促动力药 + 生活方式 | H₂ 受体拮抗剂（雷尼替丁替代） |
| ACEI 咳嗽 | 停用 ACEI，换 ARB | — |
| COPD | 戒烟；LAMA（噻托溴铵）± LABA | ICS（重度）、肺康复 |
| 百日咳 | 阿奇霉素（成人 500 mg/d × 3 天） | 隔离传染源 |

### 5.2 对症止咳（辅助）

| 药物类型 | 代表药物 | 适用场景 | 注意 |
|---------|---------|---------|------|
| 中枢性止咳药（镇咳） | 右美沙芬、可待因 | 剧烈干咳，影响睡眠；急性感染 | 可待因有成瘾风险，12岁以下禁用；有痰者慎用（会妨碍排痰） |
| 外周性止咳药 | 苯丙哌林 | 刺激性干咳 | — |
| 祛痰药（黏液溶解） | 乙酰半胱氨酸、氨溴索、厄多司坦 | 痰多、痰稠难以咳出 | 促进排痰，不止咳 |
| 复方制剂 | 复方甘草合剂、含蜂蜜的咳嗽糖浆 | 轻症对症 | 注意含酒精/糖成分 |

> ⚠️ **止咳药使用原则**：有痰的咳嗽不宜单用止咳药，应先祛痰；慢性咳嗽优先找病因，而非长期依赖止咳药掩盖症状。

### 5.3 难治性慢性咳嗽（Refractory Chronic Cough）

当常规排查和治疗均无效时，考虑"**咳嗽超敏综合征（Cough Hypersensitivity Syndrome，CHS）**"：

- **机制**：咳嗽感受器（TRPV1、TRPA1、P2X3 受体）神经敏化，轻微刺激即触发强烈咳嗽
- **治疗选项**：
  - **神经调节药物**：加巴喷丁（100–300 mg/d 起）、阿米替林（低剂量 10–25 mg/d 睡前）
  - **语言/行为治疗（Speech and Language Therapy）**：有循证支持，通过呼吸控制技术减少咳嗽
  - **新型靶向药**：P2X3 受体拮抗剂（如 Gefapixant，已在多国上市）——专门针对咳嗽超敏，有效率约 18–20%，主要副作用为味觉改变
  - **心理治疗**：针对有焦虑/习惯性成分的功能性咳嗽

## 六、生活方式干预

| 干预措施 | 适用场景 | 效果 |
|---------|---------|------|
| **戒烟** | 所有因吸烟所致或加重的咳嗽 | 停止吸烟后 4–8 周咳嗽显著减轻 |
| **充足水分** | 所有类型咳嗽，尤其感染后 | 稀化痰液，促进排出 |
| **空气加湿** | 干燥环境诱发的刺激性干咳 | 减少气道黏膜干燥刺激 |
| **避免过敏原** | 过敏性鼻炎、哮喘相关咳嗽 | 控制接触是根本预防 |
| **床头抬高** | GERD 相关夜间咳嗽 | 抬高 15–20 cm 减少夜间反流 |
| **减重** | 肥胖伴 GERD 或睡眠呼吸暂停所致咳嗽 | BMI 每降低 1，GERD 症状改善 |
| **口罩防护** | 职业性粉尘/化学品刺激 | 必要时更换工作环境 |
| **蒸汽吸入** | 上呼吸道感染所致黏稠分泌物 | 辅助稀化分泌物 |

## 七、特殊人群注意事项

### 7.1 儿童

- 主要额外病因：异物吸入（尤其 1–3 岁）、迁延性细菌性支气管炎（PBB）
- 哮喘比例高：儿童慢性咳嗽中哮喘占 30–40%
- **过度诊断风险**：许多儿童反复咳嗽被误诊为"反复感冒"，实为过敏性哮喘或鼻炎未处理
- 年龄限制：可待因（中枢镇咳）< 12 岁禁用；右美沙芬需慎用于幼儿

### 7.2 老年人

- COPD、心功能不全、多种药物（ACEI）是主要额外负担
- 肺癌风险随年龄增加；> 40 岁有吸烟史的慢性咳嗽须做低剂量 CT 筛查

### 7.3 孕妇

- ACEI 类禁止在孕期使用
- 胃食管反流在妊娠后期更为普遍（腹压增加）
- 对症处理首选非药物措施；必要时与产科医生共同评估用药安全性

## 八、何时必须立即就医？（红旗症状）

以下任何一条出现，须尽快就医，不可自行处理：

- 🩸 **痰中带血**（咯血）
- 🌡 **持续高热 > 3 天**
- 😮‍💨 **呼吸困难**（静息时气促）
- ⚖️ **不明原因体重骤降**（1 个月内 > 5 kg）
- 🌙 **盗汗 + 低热 + 乏力**（结核预警）
- 🔊 **声音嘶哑持续 > 2 周**（喉癌/纵隔压迫）
- 📅 **咳嗽 > 8 周且无改善**
- 🚬 **40 岁以上 + 长期重度吸烟史**（肺癌风险）
- 💊 **免疫功能低下**（HIV/器官移植/长期激素）

## 九、核心结论

1. **急性咳嗽**（< 3 周）多为病毒感染，绝大多数自愈，抗生素无益
2. **慢性咳嗽**的三大病因（UACS/哮喘/GERD）覆盖约 90% 患者，按序逐步排查有效
3. **ACEI 类降压药**是最容易被忽视的可逆性咳嗽原因，换药即可解决
4. **亚洲人 ACEI 咳嗽发生率**（30–40%）远高于西方人，服此类药的患者尤需注意
5. **咳嗽变异型哮喘**常被误诊为感冒残留咳嗽，关键线索是夜间/凌晨加重、运动或冷空气诱发
6. **难治性咳嗽**可能是"咳嗽超敏综合征"，单纯止咳药无效，需神经调节药物或行为治疗
7. **止咳药是辅助，不是解决方案**——找病因才是根本，长期靠止咳药是在掩盖问题
