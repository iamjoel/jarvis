# 中国非物质文化遗产

> 非物质文化遗产（Intangible Cultural Heritage，简称"非遗"）是指各族人民世代相传，并视为其文化遗产组成部分的各种传统文化表现形式，以及与传统文化表现形式相关的实物和场所。

中国是世界上非物质文化遗产最为丰富的国家之一。国务院自 2006 年起，先后公布了五批国家级非物质文化遗产代表性项目名录，共计 1557 项（含扩展项目）。这些项目涵盖了中华民族数千年来积淀的民间智慧、艺术审美与生活哲学。

## 申遗成就

截至 2024 年，中国共有 **44 个项目**列入联合国教科文组织非物质文化遗产名录，居世界首位。包括：

- **人类非物质文化遗产代表作名录**：昆曲、古琴艺术、新疆维吾尔木卡姆艺术、蒙古族长调民歌、中国篆刻、中国雕版印刷技艺、中国书法、中国剪纸、中国传统木结构建筑营造技艺、中国端午节、中国朝鲜族农乐舞、中国妈祖信俗、南音、热贡艺术、中国传统桑蚕丝织技艺、藏戏、玛纳斯、花儿、侗族大歌、格萨尔、京剧、中医针灸、皮影戏、中国珠算、中国二十四节气、藏医药浴法、太极拳、送王船、中国传统制茶技艺及其相关习俗、羌年、黎族传统纺染织绣技艺、中国木版雕版印刷技艺（扩展）、中国书法（扩展）等
- **急需保护的非物质文化遗产名录**：羌年、黎族传统纺染织绣技艺、中国木拱桥传统营造技艺等

## 保护体系

```
国家级非遗名录
    ↓
省级非遗名录
    ↓
市级非遗名录
    ↓
县级非遗名录
```

中国建立了四级非遗保护体系，同时设立**非遗传承人**制度，通过命名国家级代表性传承人来保障活态传承。截至 2024 年，国家级非遗代表性传承人已达 3068 人。
