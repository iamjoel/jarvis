# 中国各省份代表美食荟萃分析

## 高密度摘要

**一句话结论**：这组文档按省级行政区整理中国各地代表美食，核心目标不是追逐网红店，而是找出长期在地、反复被地方资料和本地消费验证的食物。
**核心机制**：使用类荟萃分析方法，把官方/非遗/地方志/地方媒体/老字号/本地日常消费多源交叉，并对景区、短视频、伴手礼和商业化偏倚降权。
**判断入口**：先从省份进入，再看城市表；每个城市优先看 A/B 级且避坑提示较少的品类。
**常见误区**：名气大、排队长、平台热度高，不等于真实好吃；很多地方真正稳定的食物在早市、居民区、县城老店和宴席体系里。
**相关文档**：[荟萃分析方法](../m/meta-analysis.md)、[山西代表性面食](../s/shanxi-noodles.md)、[胡辣汤](../h/hulatang.md)、[洛阳水席](../l/luoyang-shuixi.md)。

---

## 方法说明

这不是严格统计意义上的医学荟萃分析，因为美食没有统一效应量。这里借用荟萃分析的精神：多来源纳入、偏倚识别、异质性解释、敏感性判断。每个省级行政区都按“概览 + 城市代表美食 + 网红避坑 + 来源”组织。
