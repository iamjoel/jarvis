# 动物流

## 高密度摘要

- **一句话结论**：动物流是一种以地面四足移动为核心的自重训练体系，适合想把灵活性、平衡感、核心控制、协调和趣味性结合起来的人。
- **核心机制**：通过手脚支撑、爬行、转体、髋肩活动、身体切换和连续流动序列，把关节活动度、力量、稳定和运动控制放在同一个动作链里训练。
- **判断入口**：如果你觉得瑜伽太静、普通拉伸太无聊、力量训练太器械化，又愿意在地面上练爬行、支撑和旋转，动物流值得试。
- **常见误区**：不要一开始模仿高难度视频；不要忽视手腕准备；不要把它当纯拉伸；肩、腕、肘疼痛或核心控制很弱时要从低强度版本开始。

## 它是什么

动物流（Animal Flow）是由 Mike Fitch 创建的地面自重训练系统。它看起来像瑜伽、街舞、体操、爬行训练和动物动作的混合，但本质不是随便模仿动物，而是把一组基础动作按规则连接起来。

官方体系把动物流拆成六个组成部分：

1. **手腕活动准备**：手腕热身和活动准备。
2. **激活动作**：基础支撑和激活动作，例如兽式、蟹式等。
3. **特定形态拉伸**：和动物流动作形态相关的动态拉伸。
4. **行进形态**：向前、向后、侧向移动的爬行和行进动作。
5. **切换与转场**：从一个姿势切换到另一个姿势。
6. **流动序列**：把动作串成连续序列。

所以它更像“身体移动语言”：先学字母，再学词组，最后组合成一段动作。

## 主要训练什么

| 能力 | 动物流怎么练 |
|---|---|
| 手腕和肩稳定 | 长时间手撑地、爬行、转体、支撑切换 |
| 髋部活动度 | 下蹲、侧移、跨步、打开髋和旋转髋 |
| 核心控制 | 四足支撑、抗旋转、身体离地切换 |
| 协调性 | 左右手脚交叉配合，连续改变方向 |
| 平衡感 | 低重心移动、单侧支撑、动态转移 |
| 心肺和耐力 | 连续流动序列时间变长后，会明显提升代谢压力 |

它和普通拉伸最大的区别是：拉伸常常是“被动进入一个位置”，动物流则要求你在动作中主动控制位置。

## 适合谁

适合：

- 久坐、身体僵，但不想只做静态拉伸的人。
- 想练协调、平衡、灵活性和核心的人。
- 觉得健身房器械训练无聊的人。
- 喜欢像游戏、舞蹈或解谜一样学习动作的人。
- 想找一种不依赖器械、在家也能练的训练方式的人。

不太适合直接上强度的人：

- 手腕、肘、肩正在疼。
- 腰背疼痛且不知道原因。
- 膝盖无法承受深屈或跪撑。
- 完全不能稳定支撑自身体重。

这些情况不是永远不能练，而是需要从手腕准备、基础支撑、低幅度动作和更短时间开始。

## 新手怎么开始

第一次不要学一整套炫技流动序列。先学这些基础：

1. **手腕准备**：手掌前后左右轻压、掌根承重、手指方向变化。
2. **兽式**：类似四足跪姿，膝盖微离地，练核心和肩稳定。
3. **蟹式**：仰向手脚支撑，练肩伸展、臀部和后链。
4. **负载兽式**：从兽式向后坐，练髋、肩和背部延展。
5. **猿式**：低蹲侧向移动，练髋部活动度和脚踝控制。
6. **简单切换**：从兽式到蟹式，或从负载兽式到前伸。

一节新手课更合理的结构：

1. 5-10 分钟手腕、肩、髋热身。
2. 10-15 分钟学习 2-3 个基础动作。
3. 10-15 分钟练动作之间的切换。
4. 5 分钟组成一个很短的流动序列。
5. 结束后做手腕、肩、髋放松。

## 和流动式活动度训练的区别

动物流更像一套有固定语言和动作库的系统；流动式活动度训练更像一种训练方法。

| 对比 | 动物流 | 流动式活动度训练 |
|---|---|---|
| 形式 | 有品牌体系和动作分类 | 泛指流动式活动度训练 |
| 核心动作 | 兽式、蟹式、猿式、蝎式、切换等 | 髋、肩、脊柱、踝的动态活动度组合 |
| 趣味性 | 更像地面移动、舞蹈和游戏 | 更像热身、恢复和身体维护 |
| 强度 | 可低可高，手腕肩核心负担更明显 | 通常更温和，可作为日常维护 |
| 入门风险 | 手腕、肩和核心要求更高 | 更容易调整难度 |

如果你想要“好玩、像技能学习”，先试动物流。  
如果你想要“每天让身体更顺、更不僵”，先试流动式活动度训练。

## 选课时看什么

好的动物流课应该：

- 先做手腕和肩的准备。
- 明确教兽式、蟹式、猿式等基础动作。
- 给初级替代动作。
- 不一开始就做复杂倒转、跳跃和快速切换。
- 老师会纠正肩胛、骨盆、手腕承重和呼吸。

需要警惕：

- 上来就模仿复杂视频。
- 完全不做手腕准备。
- 把疼痛解释成“打开了”。
- 强行追求动作好看，而不是动作可控。
