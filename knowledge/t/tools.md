# Tools
## Cloud Service
- [Surge](https://surge.sh/) Host html/css/js pages for free. Support custom domain. Can use cli to manage.
  - Set domain: `echo your-domain.surge.sh > CNAME`
  - Deploy: `surge ./dist your-domain.surge.sh`
  - Teardown: `surge teardown`
  - List all projects: `surge list`
- Cloudflare Pages
- Netlify
- Vercel

## Design
* [Color Hunt](https://www.colorhunt.co/) Color palette inspiration.
* [Impeccable](https://impeccable.style/) 提升 UI 和 UX 的 Skill。把设计词汇封装成 Skill。
* [Stitch](https://stitch.withgoogle.com/) Google 出的 Vibe 设计图的工具。无限画布风格。
  * 支持编辑，导出。
  * 会生成设计系统( UI + Design.md)。 也能从取某个网站抓取那个网站的设计，生成设计系统。(extract the design system from xxx)。
  * 多种导出方式。 Copy as code 就是对应的 html 代码。

## Visualization
* [Carbon](https://carbon.now.sh/) Beautiful code snippets in the blink of an eye. Support many themes and languages. Can export as PNG or SVG.

## Coding
